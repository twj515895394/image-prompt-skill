# Character Assets Playbook

## 目标

生成可复用的角色资产包，而不是一次性的成片大图。

## 核心流程

1. 固定身份锚点
2. 固定外貌锚点
3. 固定发型和妆容锚点
4. 固定服装与配色锚点
5. 明确可变项
6. 规划表情组与姿态组
7. 补一致性限制

## 必须保留的东西

- 脸部识别锚点
- 发型主轮廓
- 核心配色
- 服装主识别点

## 固定项 / 可变项卡

### 固定项

- 姓名或代称
- 年龄段
- 脸型与五官比例
- 发长 / 发色 / 刘海
- 主服装轮廓
- 主色 / 辅色 / 点缀色

### 可变项

- 表情
- 姿态
- 换装
- 小道具
- 局部妆面强度

## 资产清单维度

### 身份与气质层

- 姓名或代称
- 年龄段
- 职业或身份
- 关系定位
- 性格底色
- 眼神常态

### 外貌锚点层

- 脸型
- 眉形
- 眼距与眼型
- 鼻梁高度
- 唇形
- 下巴长度
- 肤色与皮肤质感

### 发型妆容层

- 发长
- 刘海
- 发色
- 发饰
- 妆容方向
- 情绪强度对应妆面变化

### 服装配色层

- 主服装轮廓
- 领口
- 袖口
- 门襟
- 裤型或裙型
- 主色、辅色、点缀色
- 材质类型

### 细节资产层

- 鞋正侧后视图
- 手部自然态与握物态
- 饰品
- 道具
- 半身主视觉
- 全身站姿

### 表演资产层

- 常态表情
- 微笑
- 压抑
- 惊讶
- 愤怒
- 悲伤
- 习惯性小动作

## 什么时候继续下钻控制页

如果角色资产除了“一致性”之外，还明确追求下面这些方向，不要只停在资产骨架页：

- `看不出是 AI`
- `自然皮肤质感`
- `别像网红脸`
- `漂亮侧颜`
- `自然体态`
- `职场人物 / 办公室纪实感`
- `别像面瘫`
- `生活化微表情`
- `通勤穿搭`
- `靠面料和版型带线条`

优先下钻：

- `style-control/anti-ai-realism.md`
- `style-control/failure-patterns-negative.md`
- `portrait/side-profile-bodyline.md`
- `portrait/commuter-outfit-bodyline.md`
- `scene-office/office-candid-photography.md`
- `portrait/lifestyle-micro-expression.md`

使用原则：

- 人设图、半身主视觉和全身站姿最适合加载这些控制页
- 如果角色设定是职场女性、白领、律师、设计师、医生等现代人物，优先考虑办公室纪实和体态线条控制
- 如果角色需要多角度头像，先用真实感与失败模式约束锁住脸部质感，再扩表情组

## 推荐输出包

1. 三视图
2. 4-6 张多角度头像
3. 6 张基础表情
4. 半身主图
5. 全身站姿
6. 手部与鞋细节
7. 换装组

## 输出骨架

```text
任务目标：
固定项：
可变项：
角色资产 Prompt：
推荐出图顺序：
一致性限制：
```

## 常见失败

- 把成片图语言直接套到资产图
- 只写漂亮，不写一致性锚点
- 可变项与固定项边界不清
- 只出了脸，没有把手部、鞋、饰品、配色这些复用资产一起锁定

## 修复方式

- 角色总变脸：减少表情强度，补五官比例锚点
- 换装就像换人：把服装与发型边界拆开
- 资产包不全：按推荐输出包顺序回补
- 角色太像棚拍写真：补真实感、失败模式和职场/生活环境控制
