# Diagnostics Index

诊断层只用于跨任务、跨维度的综合故障分析。正常生成流程默认不读取。

- 人物、服装、道具或场景跨图漂移：`identity-consistency-failure.md`
- 人体、手部、接触或动作关系错误：`anatomy-and-contact-failure.md`
- 主体不清、画面拥挤或构图难读：`composition-readability-failure.md`
- 风格、材质、光影或媒介互相冲突：`style-and-material-conflict.md`
- 多张参考图职责和优先级冲突：`reference-conflict.md`
- Prompt 过长、重复、互斥或指令过载：`prompt-overload-and-contradiction.md`

单一领域的小问题优先使用对应叶子 Reference 的失败修复，不进入诊断层。
