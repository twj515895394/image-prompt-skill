# 角色资产 Playbook

## 用途

生成可复用、可延续、可用于多场景和多镜头的角色资产，而不是只做一张漂亮成片。

## 何时读取

适用于：

- 角色设定图
- 三视图
- 多角度头像
- 表情板
- 姿态板
- 换装板
- 手部、鞋履、饰品和道具细节
- 角色主视觉

## 核心目标

先锁定角色识别，再扩展可变资产。

```text
身份锚点
→ 外貌锚点
→ 轮廓与比例
→ 发型、服装和配色
→ 固定道具
→ 可变表情、姿态和换装
→ 一致性限制
```

## 固定项

至少判断：

- 年龄感与身份
- 脸型、五官比例和辨识特征
- 头身比、肩宽、体型和主要轮廓
- 发型主轮廓与发色
- 主服装轮廓和核心配色
- 标志性饰品或道具
- 常态眼神与气质

## 可变项

可根据资产目标变化：

- 表情
- 姿态
- 动作
- 换装
- 场景
- 小道具
- 局部妆面强度

固定项和可变项必须明确分开，避免每扩展一张资产就像换了角色。

## 推荐资产包

按需要选择，不要求每次全部生成：

1. 角色识别主图
2. 正、侧、背三视图
3. 4–6 张多角度头像
4. 基础表情组
5. 常用姿态与习惯性小动作
6. 全身中性站姿
7. 手部、鞋履、饰品和道具细节
8. 换装组
9. 特殊状态或剧情状态

## 输出形态

### 三视图

重点锁定：比例、轮廓、服装结构、正侧背一致性。

### 表情板

重点锁定：脸部稳定区、表情强度和可变化区域。

### 换装板

重点锁定：人物身份不随服装变化，发型和脸部锚点持续稳定。

### 姿态板

重点锁定：身体比例、重心、习惯动作和角色气质。

## 常见失败与修复

- 多角度总变脸：减少表情强度，补五官比例和脸部稳定区
- 换装就像换人：把服装与身份锚点分离
- 三视图比例不一致：固定头身比、肩胯宽度和服装长度关系
- 资产包只出脸：根据实际用途回补全身、手部、鞋履和道具
- 角色过度依赖场景氛围：让身份在中性背景中也成立
