# 局部图片编辑 Playbook

## 用途

对现有图片进行局部、精确、受约束的修改，并尽可能保持未指定区域不变。

## 何时读取

适用于：

- 替换单个物体
- 删除多余元素
- 增加指定人物或道具
- 修改聊天气泡或画面文字
- 调整某只手、某个姿态或局部结构
- 修复指定区域
- 保持其他人物、场景、服装、机位和光线不变

如果用户允许构图、场景、风格和整体画面大幅变化，应改用整体图生图任务。

## 核心编辑合同

每次编辑必须尽量建立以下约束：

```text
必须保留：
必须修改：
必须删除：
必须增加：
允许变化：
禁止变化：
空间位置：
数量与身份：
文字内容：
```

不要求每次都机械输出全部字段，但内部必须完成判断。

## 执行顺序

1. 识别编辑目标和目标区域
2. 列出必须保持不变的主体与背景锚点
3. 明确修改前对象与修改后对象
4. 明确位置、方向、尺度、数量和遮挡关系
5. 明确文字是否需要逐字准确
6. 补充局部材质、光影和透视匹配
7. 写出禁止出现的旧元素或错误元素

## 保持不变的常见维度

- 人物身份与五官
- 发型、服装和配色
- 人物数量与站位
- 背景场景和道具状态
- 机位、焦段和构图
- 光源方向和阴影
- 画幅比例和画面风格

用户说“其他所有保持不变”时，不能只在 Prompt 末尾写一句，而要把最关键的稳定锚点显式列出。

## 局部替换规则

替换物体时写清：

- 原物体是什么
- 新物体是什么
- 新物体放在哪里
- 大小是否沿用原物体
- 朝向、接触面和遮挡关系
- 是否保留原物体的包装、形状或轮廓
- 原物体是否必须完全消失

## 增删元素规则

### 删除

- 明确删除对象
- 明确删除后由什么背景或材质自然补全
- 防止留下残影、边缘或重复结构

### 增加

- 明确数量
- 明确位置
- 明确与现有主体的关系
- 明确光影、透视和风格必须匹配原图

## 文字编辑规则

- 用户提供的文字优先逐字保留
- 不擅自润色、改写或纠正数字
- 写清字体气质、字号层级、气泡或牌匾位置
- 除目标文字外，其他 UI、布局和图像内容保持不变

## Prompt 组装骨架

```text
基于参考图进行局部编辑。

必须保持：
[人物、场景、构图、光影、比例等稳定项]

只修改：
[目标区域和具体变化]

空间与数量要求：
[位置、尺度、方向、数量、遮挡和接触关系]

禁止：
[旧元素残留、额外人物、重复物体、非目标区域变化]
```

## 常见失败与修复

- 修改一个物体却重画整张图：强化禁止变化项和稳定锚点
- 新物体悬空或比例错误：补接触面、尺度和透视关系
- 删除后留下残影：明确自然补全原背景
- 人物身份漂移：重复锁定脸、发型、服装和站位
- 指定文字出错：减少装饰性描述，逐字给出目标文本
- 旧元素仍出现：把旧对象写入必须删除和禁止出现
