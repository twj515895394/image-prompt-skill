# 电影剧照与影视叙事画面语言

## 解决什么问题

把“电影感”从抽象标签拆成事件瞬间、摄影语言、光影、色彩、景深和情绪关系，用于生成具有叙事落点的电影剧照、影视封面和单帧故事画面。

## 什么时候读取

- 用户明确要求电影剧照、影视叙事画面或类型片视觉；
- 单张成片需要看起来像故事正在发生，而不是普通写真或概念图；
- 用户只写“电影感、高级感、导演感”，缺少可执行画面语言；
- 画面有漂亮光影但没有事件、人物关系或情绪落点；
- 需要区分生活化纪实、商业写真与电影叙事画面。

如果只需要选择景别、机位或焦段，读取 `../../controls/composition-camera/shot-angle-lens-selection.md`；如果只需要选择色调，读取 `../../libraries/lighting-color/cinematic-color-palettes.md`。

## 核心结构

```text
电影画面
= 清晰主体
+ 可信场景
+ 正在发生的事件瞬间
+ 有动机的摄影位置
+ 有来源的光影
+ 统一色彩关系
+ 可读的情绪落点
```

删掉“电影感”三个字后，剩余描述仍应能够成立。

## 事件瞬间

电影剧照优先选择动作或关系过程中的一个节点：

- 行动开始前的停顿；
- 视线刚发生变化；
- 道具即将被拿起、递出或落下；
- 人物刚进入或准备离开空间；
- 冲突爆发前一秒；
- 动作完成后的反应；
- 对话中隐藏情绪被看见的瞬间。

避免只写人物站在场景中摆姿势。

## 题材级视觉方向

| 方向 | 核心视觉关系 | 适合题材 | 避坑 |
|---|---|---|---|
| 都市孤独现实 | 玻璃反射、夜景、长焦压缩、人物间距离 | 都市情绪、暧昧、夜行 | 不要把少量霓虹扩成赛博世界 |
| 家庭克制现实 | 现场自然光、日常空间、稳定镜头、收敛表演 | 家庭、成长、生活流 | 减少炫技构图和过强轮廓光 |
| 权力史诗与仪式 | 低角度、对称秩序、群像调度、大色块 | 权谋、典礼、登场 | 不要只剩宏大空景和装饰 |
| 废墟哲思 | 长镜头感、水面或空间反射、低饱和、时间痕迹 | 末世、回忆、意识流 | 节奏与空间必须支持缓慢观察 |
| 群像戏剧与对峙 | 固定空间、明确 blocking、视线交叉、反差光 | 帮派、武士、团队冲突 | 先站位再决定镜头，不要随机排人 |
| 奇幻史诗 | 写实材质、巨构尺度、文明遗迹、人物尺度参照 | 奇幻、世界观、冒险 | 先写世界规则，不堆魔法名词 |
| 秩序怪诞 | 对称布景、人工色块、冷静机位、表演反差 | 黑色幽默、寓言、广告 | 道具、演员和空间必须服从同一秩序 |

## 摄影语言

电影画面中的镜头选择必须服务关系：

- 广角：让人物被空间包围，或强化动作与环境；
- 标准视角：保持自然关系，适合克制对白和现实题材；
- 长焦：压缩人物距离，制造观察、孤独或暧昧；
- 过肩与反打：建立双方关系和信息差；
- 前景遮挡与反射：增加观察感、隐秘感和空间层次；
- 对称构图：表现秩序、权力、仪式或怪诞；
- 负空间：表现等待、疏离、未知或人物去向。

不要使用导演名替代这些可见变量。

## 光影与色彩

- 光线必须能从窗户、灯具、天空、火焰、霓虹或环境反射中找到依据；
- 阴影、高光和主体分离应服务同一情绪；
- 人物戏优先保护眼神、脸部识别和关系距离；
- 场景戏优先保护前中后景和尺度；
- 色调由阴影色、高光色和主体保留色构成，不只是全局滤镜；
- 强类型色调只使用一个主方向。

详细控制：

- 灯光：`../../controls/lighting-color/cinematic-lighting-patterns.md`
- 色调：`../../libraries/lighting-color/cinematic-color-palettes.md`

## 景深和材质

- 浅景深适合情绪特写，但要保留必要环境证据；
- 中等景深适合人物关系和短剧场景；
- 深景深适合群像、空间叙事和史诗场面；
- 皮肤、布料、金属、玻璃、墙面和空气颗粒都应符合场景光源；
- 电影质感不等于全图颗粒、黑边、横向光晕和低曝光叠加。

## Prompt 结构

```text
[主体] 在 [可信场景] 中正处于 [事件瞬间]，
观众从 [景别、机位与观看关系] 观察，
[焦段和空间关系] 强化 [人物关系或环境压力]，
场景由 [合理光源] 照明，
[阴影、高光与主体分离] 服务 [情绪]，
整体采用 [单一色调方向]，
保留 [关键材质、环境证据和叙事道具]，
画面像故事进行中的电影剧照，而不是摆拍写真。
```

## 与相邻风格的边界

### 与生活化纪实摄影

- 纪实摄影强调真实在场、随手记录和合理不完美；
- 电影剧照强调叙事选择、情绪落点和更明确的视听组织；
- 两者可共享自然光和克制表演，但不默认互相替代。

### 与商业写真

- 商业写真优先人物或产品展示完成度；
- 电影剧照优先事件、关系和故事张力；
- 只有漂亮人物、棚拍光和背景虚化，不足以构成电影叙事画面。

### 与概念艺术

- 概念艺术优先设计、世界观和功能；
- 电影剧照优先某个可被镜头记录的具体瞬间。

## 常见失败与修复

- 画面像普通写真：增加事件瞬间、视线关系和叙事道具；
- 只剩黑暗和颗粒：恢复可信曝光、材质和空间层次；
- 风格词很多但没有摄影位置：明确观众从哪里、以什么距离观察；
- 光影漂亮但不可信：补可见或可推断的场景光源；
- 人物和背景各自好看但无关系：建立站位、视线、动作或遮挡；
- 类型风格互相混杂：保留一个题材级视觉方向，其余作为弱辅助；
- 导演名主导 Prompt：删除姓名，改写成构图、镜头、光影、色彩和节奏变量。
