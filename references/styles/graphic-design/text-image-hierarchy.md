# 文字与图像层级系统

## 解决什么问题

建立海报、封面、广告和品牌视觉中的文字层级、图像主次和阅读顺序，避免标题、主体、卖点和装饰互相争抢。

## 什么时候读取

- 海报、活动主视觉、缩略图、产品广告和社交媒体宣传图；
- 画面需要标题、副标题、卖点、品牌和行动信息；
- 用户要求“文字清楚”“排版高级”“标题有冲击力”；
- 文字与人物、产品或背景发生遮挡；
- 图像生成中需要明确文字区域和准确性要求。

## 信息层级

默认从高到低组织：

```text
主标题
→ 主视觉
→ 副标题 / 核心卖点
→ 时间地点 / 数据 / 辅助说明
→ 品牌 / Logo / 行动信息
```

具体任务可以让主视觉高于主标题，但必须明确谁是第一视觉落点。

## 文字层级字段

- 内容角色：标题、副标题、标签、正文、数据、品牌；
- 字数与行数；
- 字号关系；
- 字重；
- 对齐方式；
- 行距和字距；
- 颜色与背景对比；
- 是否允许描边、底色块或阴影；
- 与主体的遮挡边界；
- 阅读顺序。

## 主标题

- 只表达一个核心主题；
- 字数受控，避免长句变成小字；
- 可以通过尺寸、字重、颜色、位置或与图像穿插建立冲击力；
- 不应遮挡人物五官、产品标签和关键动作；
- 标题字形可以风格化，但可读性优先。

## 副标题与辅助信息

- 副标题解释主标题，不重复同一句话；
- 时间、地点、规格、价格和卖点应分组；
- 辅助信息与主标题形成明显字号差；
- 小字区域需要更平整的背景；
- 信息较多时使用网格、卡片或分栏，不在一处堆叠。

## 图像与文字关系

### 图像主导

适合电影、人物、产品和旅行宣传：

- 主体先被看见；
- 文字利用负空间；
- 标题不破坏主体轮廓和视线；
- 装饰图形只辅助视觉动线。

### 文字主导

适合字体海报、促销、音乐和潮流视觉：

- 标题成为主要图形；
- 图像作为背景、纹理或穿插元素；
- 仍需保留文字识别；
- 不应把所有字都做成同等强度。

### 图文并重

适合品牌提案、产品介绍和教育宣传：

- 使用双栏、网格或模块；
- 明确阅读起点和结束点；
- 统一间距、对齐和圆角等系统；
- 模块之间有主次，不平均用力。

## 对比方式

优先从以下方式选择两到三种：

- 大小对比；
- 字重对比；
- 明度对比；
- 色相对比；
- 疏密对比；
- 位置对比；
- 字体风格对比；
- 图像清晰度与背景简化。

不要同时使用过多描边、发光、渐变、阴影和立体字效果。

## 中文、英文与数字

- 中文标题优先控制字数、字重和行距；
- 英文标题注意大小写、字距和断行；
- 数字适合强化价格、年份、步骤和核心指标；
- 中英混排需统一基线和视觉重量；
- 不要为“高级感”随机加入无意义英文。

## 文字准确性

图像生成任务中应明确：

- 必须出现的准确文字；
- 语言和大小写；
- 允许换行的位置；
- 禁止出现的多余文字；
- Logo 是否为图形标识或普通文本；
- 文字错误是否需要后续编辑流程修复。

排版规则不能保证模型一次生成完全准确的长文本；长正文和复杂数据应优先使用后期排版或局部文字编辑。

## Prompt 结构

```text
建立清楚图文层级：
第一视觉落点为 [主标题 / 主体]，
主标题内容为“[准确文字]”，位于 [位置]，使用 [字号、字重和对齐]；
副标题位于 [位置]，与主标题形成明显层级；
品牌和辅助信息放在 [安全区]；
文字与背景保持 [明度 / 色相 / 底色块] 对比，
不得遮挡 [五官 / 产品标签 / 关键动作]，
阅读顺序为 [顺序]，禁止无意义装饰文字。
```

## 常见失败与修复

- 所有文字同样大：建立主标题、副标题和辅助信息三级关系；
- 标题难读：简化背景、提高对比或加入受控底色块；
- 字体效果太多：保留一种主效果，其余回归简洁；
- 标题遮挡主体：重建负空间和遮挡边界；
- 小字大量乱码：减少文字量或改用后期排版；
- 无意义英文泛滥：只保留有业务含义的文字。
