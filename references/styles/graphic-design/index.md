# Graphic Design Styles

当任务需要海报、品牌、包装、版式、信息图、UI 视觉或平面宣传设计时读取本分类。

## 文字与图像层级

当海报、封面、缩略图、广告或品牌视觉需要组织主标题、主视觉、副标题、卖点、品牌和行动信息，并控制文字可读性与遮挡关系时，读取：

- `text-image-hierarchy.md`

## 信息图视觉语言

当任务需要流程、时间线、对比、层级、分类、数据指标、结构示意或社交长图，并需要让视觉结构匹配信息关系时，读取：

- `infographic-visual-language.md`

## 边界

- 海报、封面和缩略图的版式类型进入 `../../libraries/composition-shot/poster-layout-types.md`；
- 本目录保存排版、层级、网格、文字与图形关系的实现方式；
- 具体产品、素材、数据和题材选项进入 libraries；
- 精确长文本和复杂数据图表优先后期排版或局部文字编辑；
- UI 视觉后续按独立真实需求处理。
