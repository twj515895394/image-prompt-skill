# 环境人像与编辑写真风格

## 解决什么问题

定义一种介于生活抓拍与电影剧照之间的人像摄影：人物是明确主角，姿态和服装可以经过适度引导，环境用于说明身份、职业、旅行、文化或生活方式，但画面不必依赖电影剧情事件。

## 什么时候读取

- 用户要求环境人像、杂志写真、旅行人像、职业肖像或生活方式大片；
- 人物需要与街道、建筑、室内、自然环境共同表达主题；
- 用户需要比生活抓拍更精致，但不想要电影剧情和强戏剧感；
- 商业人物写真需要保留真实环境，而不是纯棚拍背景；
- 图像需要适度摆姿、服装设计和画面控制。

## 核心公式

```text
人物身份与气质
+ 有意义的环境
+ 适度引导姿态
+ 可信现场或模拟现场光
+ 清楚人物—环境层级
+ 编辑级完成度
```

## 与相邻风格的边界

### 与生活化纪实摄影

- 纪实摄影强调顺手记录、动作正在发生和合理不完美；
- 环境人像允许人物配合镜头、停留和调整姿态；
- 纪实摄影像关系中的真实片刻；
- 环境人像像杂志、旅行专题或职业人物报道中的成片。

### 与电影剧照

- 电影剧照优先事件、冲突、关系和剧情瞬间；
- 环境人像优先人物身份、服装、姿态与地点关系；
- 环境人像可以有故事气息，但不要求明确前后剧情；
- 需要强叙事事件时读取 `../cinematic/film-still-language.md`。

### 与纯棚拍写真

- 环境人像保留地点证据、空间层次和现场材质；
- 纯棚拍更依赖背景纸、布景和人工灯位；
- 环境不能只是虚化色块，应能解释人物为何出现在这里。

## 人物与环境关系

环境应承担至少一种功能：

- 身份：工坊、办公室、舞台、厨房、实验室；
- 地域：老街、海边、山地、车站、异国建筑；
- 生活方式：咖啡馆、书店、工作室、运动场；
- 文化：传统建筑、服饰、节庆、手工空间；
- 情绪：空旷广场、雨后街道、窗边、荒野；
- 视觉结构：门框、窗户、楼梯、拱门、道路和前景遮挡。

环境必须支持人物，不应成为无关旅游背景。

## 姿态与表演

- 允许人物正对、侧对或回望镜头；
- 姿态应与身份和环境一致；
- 手部可以接触栏杆、工具、衣领、包、桌面或环境道具；
- 重心和视线要自然，不使用无目的标准影楼姿势；
- 服装展示可以明确，但不能让人物像商品假人；
- 表情通常克制、稳定或轻微互动，不必持续微笑。

## 构图与镜头

- 中景和全身适合表现人物与环境；
- 中近景适合兼顾面部与地点证据；
- 广角用于空间感时必须保护人物比例；
- 长焦可压缩街景或建筑层次，但不要抹掉环境意义；
- 门框、玻璃、植物、车辆或建筑线条可形成前景和引导线；
- 背景保持可读，但明度、清晰度或颜色权重低于人物。

## 光线

- 优先使用窗光、阴天漫射光、日落、街灯、店铺灯或环境反射；
- 可以补光或控制反差，但最终仍应像地点中的真实光线；
- 人物脸部、服装和环境材质处于同一光照逻辑；
- 避免人物像被单独抠图后贴到背景；
- 强轮廓光和烟雾只在主题确实需要时使用。

## 色彩与材质

- 人物服装与环境形成协调、对比或呼应关系；
- 保留墙面、木材、石材、金属、植物和天气痕迹；
- 肤色不被环境色完全污染；
- 编辑感来自统一关系，不来自全局滤镜和过度磨皮；
- 题材可以时尚、旅行、职业或文化，但只保留一个主视觉方向。

## Prompt 结构

```text
环境人像 / 编辑写真，人物为 [身份与气质]，
位于 [具有主题意义的真实环境]，
环境通过 [建筑、道具、材质或地域证据] 说明 [身份 / 旅行 / 生活方式 / 文化主题]。
人物采用 [适度引导姿态]，视线 [方向]，手部与 [环境元素] 自然互动。
使用 [景别、机位和焦段感] 同时保护人物与地点关系，
光线来自 [窗户 / 天空 / 街灯 / 店铺 / 环境反射]，
人物和环境处于统一光照与色彩系统，
画面具有杂志编辑完成度，但不是随手抓拍、纯棚拍或剧情化电影剧照。
```

## 常见失败与修复

- 画面像普通旅游打卡：让环境明确服务人物身份或主题；
- 人物像贴在背景上：统一光向、色温、接触和景深；
- 环境被虚化成色块：恢复可识别地点证据；
- 姿势像影楼模板：让手部、重心和视线与环境发生关系；
- 漂成随手抓拍：提高服装、构图和姿态控制；
- 漂成电影剧照：减少强剧情事件和类型化冲突；
- 环境抢人物：降低背景对比、饱和度或清晰度。
