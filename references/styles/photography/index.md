# Photography Styles

当任务需要真实摄影、手机抓拍、纪实、环境人像、编辑写真、产品或其他照片质感时读取本分类。

## 生活化纪实摄影

当用户要求手机抓拍、同事或朋友视角、伴侣视角、非摆拍日常、普通现场光、不过度精修，或“女友感 / 男友视角”这类亲近关系生活写真时，读取：

- `lifestyle-candid-photography.md`

该叶子统一覆盖普通生活纪实与亲近关系视角；不为“女友感”复制第二份风格正文。

## 环境人像与编辑写真

当用户要求杂志写真、旅行人像、职业肖像、生活方式大片，允许适度引导摆姿，并需要人物与真实地点共同表达身份、文化或气质时，读取：

- `environmental-editorial-portrait.md`

它位于生活抓拍与电影剧照之间：比抓拍更受控，但不要求明确剧情事件。

## 商业产品摄影

当用户要求电商主图、品牌产品广告、食品饮料广告、奢侈品或科技产品的真实摄影表现，并需要准确形态、材质、棚拍光、背景和商业留白时，读取：

- `product-photography.md`

## 边界

- 具体人物、场景、服装、产品和事件选项进入 `libraries/`；
- 构图、机位、光线、姿态和微表情判断进入 `controls/`；
- 明确剧情事件和类型片叙事进入 `../cinematic/`；
- 结构展示、爆炸图和不可能机位的 CGI 产品图进入 `../3d-rendering/`；
- 本目录只说明摄影风格如何呈现；
- 纯棚拍人像与高时装摄影后续按真实需求补充。
