# Photography Styles

当任务需要真实摄影、手机抓拍、纪实、棚拍、时尚、人像、产品或其他照片质感时读取本分类。

## 生活化纪实摄影

当用户要求手机抓拍、同事或朋友视角、伴侣视角、非摆拍日常、普通现场光、不过度精修，或“女友感 / 男友视角”这类亲近关系生活写真时，读取：

- `lifestyle-candid-photography.md`

该叶子统一覆盖普通生活纪实与亲近关系视角；不为“女友感”复制第二份风格正文。

## 边界

- 具体人物、场景、服装和事件选项进入 `libraries/`；
- 构图、机位、光线和微表情判断进入 `controls/`；
- 电影剧照进入 `../cinematic/`；
- 本目录只说明摄影风格如何呈现；
- 棚拍、人像、产品和时尚摄影资料后续按真实需求及外部案例补充。
