# 生活化纪实摄影风格

## 用途

定义生活化、非摆拍、可被真实手机或相机记录到的摄影表现方式。

## 何时读取

- 用户明确要手机抓拍、同事视角、朋友视角或日常纪实
- 普通生活与职场场景需要降低广告感和棚拍感
- 图生图需要把精修写真改为自然生活化照片

## 风格特征

- 主体处于事件或动作中
- 镜头保持合理在场关系
- 构图允许轻微偏移、裁切或遮挡
- 环境保留真实使用痕迹
- 皮肤和布料不过度精修
- 光线来自现场，而不是为了人物单独搭建
- 色彩接近普通环境下的真实观感

## 与商业写真区分

生活化纪实摄影不强调：

- 完美居中
- 精心摆姿
- 无瑕皮肤
- 强轮廓光
- 极端景深
- 广告级背景洁净度

它强调的是“这个瞬间确实可能被身边的人拍到”。

## Prompt 写法

```text
生活化纪实摄影，像熟悉人物的同事或朋友在真实环境中顺手记录，主体没有刻意配合镜头，动作和情绪正在发生，保留自然皮肤与布料纹理、普通现场光、轻微不完美构图和可识别的环境证据，清楚但不过度精修
```

## 适合场景

- 办公室、茶水间、会议室、走廊
- 家庭厨房、客厅、阳台
- 街头、咖啡店、商场和通勤途中
- 朋友聚会和日常互动

## 风格漂移与修复

- 漂成低清废片：恢复主体清晰度，只保留自然曝光和构图偏差
- 漂成偷拍感：明确合理的人际关系和拍摄位置
- 漂成商业写真：减少摆姿、完美光和过度背景虚化
- 漂成普通快照而无重点：保留第一视觉落点和事件瞬间
