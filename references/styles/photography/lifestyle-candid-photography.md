# 生活化纪实摄影风格

## 用途

定义生活化、非摆拍、可被真实手机或相机记录到的摄影表现方式，覆盖朋友、同事、伴侣和熟悉人物之间的日常观察视角。

## 何时读取

- 用户明确要手机抓拍、同事视角、朋友视角、伴侣视角或日常纪实；
- 普通生活与职场场景需要降低广告感和棚拍感；
- 图生图需要把精修写真改为自然生活化照片；
- 用户要求“女友感”“男友视角”“像熟悉的人随手拍到”；
- 人物写真需要通过日常事件和关系距离建立亲近感，而不是只靠摆姿。

## 核心公式

```text
可信关系视角
+ 正在发生的日常事件
+ 合理拍摄距离
+ 轻动作和微表情
+ 现场光
+ 可识别环境证据
+ 受控的不完美
```

“女友感”或“男友视角”不是固定人物类型，而是一种亲近关系下的观看方式。

## 风格特征

- 主体处于事件或动作中；
- 镜头保持合理在场关系；
- 构图允许轻微偏移、裁切或遮挡；
- 环境保留真实使用痕迹；
- 皮肤和布料不过度精修；
- 光线来自现场，而不是为了人物单独搭建；
- 色彩接近普通环境下的真实观感；
- 主体知道身边有人，但不一定正面对镜头表演；
- 画面像一段关系中的真实片刻，而不是陌生人远距离偷拍。

## 关系视角

### 同事或朋友视角

- 拍摄距离偏公共和自然；
- 常见于工位、茶水间、聚会、通勤和街头；
- 动作通常来自工作、聊天、吃东西、整理物品或等待；
- 不需要过强暧昧和持续凝视镜头。

### 伴侣或亲近关系视角

- 拍摄距离更近，但必须符合场景；
- 可以出现轻微回望、递东西、等候、整理头发、分享食物、走在前方回头等动作；
- 亲近感来自互动、距离和熟悉感，不来自强制性感摆姿；
- 镜头可以略低、略偏、靠近桌边、床边、车内或同行位置；
- 不能漂成偷窥、偷拍或陌生人凝视。

## 日常事件选择

优先选择能自然带出人物状态的事件：

- 正在喝咖啡、吃早餐或分享食物；
- 走在街上、进入商店、等车或回头确认同伴；
- 整理头发、衣袖、包带、耳机或桌面；
- 看手机、读菜单、拿东西、递东西或打开门；
- 做饭、洗水果、收衣服、给植物浇水；
- 工作间隙发呆、伸懒腰、回答同事或等待电梯；
- 刚笑过、刚准备说话、被叫住后回头。

同一张图只保留一个主要事件，不要堆叠多个生活动作。

## 动作和表情

- 动作应处于开始、进行中或收尾，不是标准定格摆姿；
- 视线可以落在手中物、同行者、窗外或画外信息；
- 微表情优先克制、自然和正在发生；
- 身体重心、肩颈、手指和下巴应参与当前动作；
- 亲近关系画面可以有轻微回望或笑意，但不要每张都直视镜头甜笑。

需要具体微表情时读取 `../../controls/pose-action/lifestyle-micro-expression.md`。

## 光线和色彩

- 窗边自然光、阴天漫射光、普通室内灯、街灯和车内环境光优先；
- 混合光允许存在，但肤色和主体仍需可读；
- 不默认使用棚拍轮廓光、极端高反差和全局梦幻柔光；
- 色彩可以轻微偏暖、清透或低饱和，但必须保留真实环境颜色；
- 夏日、旅行和亲近关系场景可以更明亮，但不等于过曝和高饱和滤镜。

## 构图和画幅

- 允许轻微偏心、边缘裁切、前景遮挡和手机透视；
- 主体仍应有清楚第一视觉落点；
- 环境需要保留足够证据，不能只剩背景虚化色块；
- 竖屏 3:4 或 4:5：适合人物、社交平台和生活写真；
- 竖屏 9:16：适合移动端封面和短视频参考帧，但注意上下留白；
- 横屏 16:9：适合双人关系、旅行、街景和更完整环境；
- 正方形 1:1：适合社交头像或单事件快照，但容易压缩环境信息。

画幅只影响空间组织，不改变风格核心。

## 与环境人像区分

- 生活化纪实强调事件正在发生、人物没有刻意配合和合理构图偏差；
- 环境人像允许人物配合镜头、停留、回望和调整姿态；
- 生活化纪实像熟悉的人顺手记录；
- 环境人像像旅行专题、职业肖像或杂志编辑写真；
- 用户要求“精致写真但保留真实地点”时，读取 `environmental-editorial-portrait.md`。

## 与商业写真区分

生活化纪实摄影不强调：

- 完美居中；
- 精心摆姿；
- 无瑕皮肤；
- 强轮廓光；
- 极端景深；
- 广告级背景洁净度；
- 每张图都持续看镜头；
- 服装、妆容和身体展示压过事件本身。

它强调的是“这个瞬间确实可能被身边的人记录到”。

## 与电影剧照区分

- 生活化纪实优先真实在场、普通光线和合理不完美；
- 电影剧照优先叙事选择、类型气质和更明确的视听组织；
- 日常画面可以有故事，但不默认加载电影光影和强类型色调；
- 明确要求电影剧照时，读取 `../cinematic/film-still-language.md`。

## Prompt 写法

```text
生活化纪实摄影，像 [朋友 / 同事 / 伴侣] 在 [真实场景] 中顺手记录，
人物正在 [单一日常事件]，没有刻意摆姿，
视线落在 [互动对象 / 手中动作 / 画外信息]，
保留轻动作和克制微表情，
使用 [现场自然光或普通环境光]，
皮肤、头发和布料纹理自然，
构图允许 [轻微偏移 / 合理裁切 / 前景遮挡]，
环境仍清楚可识别，像关系中真实发生的一刻，
而不是棚拍广告、环境编辑写真或远距离偷拍。
```

## 适合场景

- 办公室、茶水间、会议室、走廊；
- 家庭厨房、客厅、阳台、卧室边缘和餐桌；
- 街头、咖啡店、商场和通勤途中；
- 公园、旅行、海边、车内和便利店；
- 朋友聚会、伴侣同行和日常互动。

## 风格漂移与修复

- 漂成低清废片：恢复主体清晰度，只保留自然曝光和构图偏差；
- 漂成偷拍感：明确熟悉的人际关系和合理拍摄位置；
- 漂成商业写真：减少摆姿、完美光和过度背景虚化；
- 漂成环境编辑写真：恢复正在发生的事件和非配合状态；
- 漂成普通快照而无重点：保留第一视觉落点和事件瞬间；
- 亲近感变成强制性感摆拍：回到日常事件、互动和观看距离；
- 每张都直视镜头：让视线服务动作或关系；
- 环境被虚化掉：恢复可识别场景证据；
- 手机感被误做成失焦噪点：保持主体清楚，只保留合理不完美。
