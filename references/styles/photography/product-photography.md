# 商业产品摄影风格

## 解决什么问题

定义真实商业产品摄影如何呈现：产品形态准确、材质可信、构图清楚、光影服务商品、背景和辅助元素不过度抢戏。

## 什么时候读取

- 电商主图、品牌产品广告、食品饮料广告、奢侈品或科技产品视觉；
- 用户要求“商业产品摄影、棚拍、广告级产品图”；
- 需要把普通产品照片改造成高完成度商业图；
- 需要区分真实摄影与 3D 产品渲染。

## 核心公式

```text
产品形态保真
+ 清楚 Hero 构图
+ 材质对应灯光
+ 可信阴影与接触
+ 受控背景和辅助元素
+ 品牌与文字安全区
```

## 风格特征

- 产品轮廓、比例、包装和配件准确；
- 第一视觉落点明确；
- 背景洁净但不必总是纯白；
- 高光、阴影和反射与材质一致；
- 产品与地面、台座或悬浮空间关系可信；
- 画面具有商业完成度，但不过度依赖“8K、masterpiece”等空泛质量词；
- 辅助元素用于解释口味、功能、场景或品牌气质；
- 文字与 Logo 区保留稳定负空间。

## 主要方向

### 干净电商主图

- 形态和标签优先；
- 背景简洁；
- 柔和棚拍光；
- 轻接触阴影；
- 禁止多余装饰和形态漂移。

### 极简品牌主视觉

- 单品或少量组合；
- 纯色、渐变或几何背景；
- 清楚负空间；
- 产品主色和品牌色稳定；
- 构图更具设计感，但不牺牲识别。

### 奢侈品商业摄影

- 深色或低饱和背景；
- 方向明确的高光和轮廓光；
- 金属、玻璃、皮革、巧克力或珠宝材质细节；
- 台座、石材、玻璃或镜面反射受控；
- 暗部仍保留纹理。

### 动态饮料与能量广告

- 产品保持最大视觉权重；
- 液体、果片、冰块、火花沿单一方向组织；
- 冻结瞬间清楚；
- 罐体或瓶体材质、标签和轮廓不被遮挡；
- 彩色轮廓光服务口味或品牌主色。

### 食品广告

- 保持菜品、甜点或零食的结构可识别；
- 重点表现切面、酥脆、融化、蒸汽、油光或湿润度；
- 配料用于解释口味；
- 不把食物变成塑料模型；
- 预留品牌和卖点区。

### 场景化生活方式产品图

- 产品处于可信使用环境；
- 场景支持产品价值，如旅行、居家、户外或办公；
- 环境复杂度低于产品；
- 产品尺寸和功能关系合理；
- 不漂成纯生活照。

## 与 3D 产品可视化的边界

- 产品摄影强调像真实相机拍摄到的物体、光线和接触；
- 3D 可视化允许悬浮、爆炸图、透明剖面和更理想化的材质控制；
- 写实 CGI 可以接近摄影，但应明确是建模渲染还是实拍风格；
- 结构展示、内部部件和不可能机位优先读取 `../3d-rendering/product-visualization.md`。

## Prompt 结构

```text
商业产品摄影，准确呈现 [产品名称、结构和材质]，
采用 [展示类型与 Hero 构图]，
从 [角度] 展示 [关键卖点或结构]，
使用与 [材质] 匹配的 [棚拍灯光]，
产品与 [地面 / 台座 / 环境] 有可信接触、阴影和反射，
辅助元素只用于说明 [口味 / 功能 / 使用场景]，
在 [位置] 保留品牌或文案安全区，
背景不过度抢主体，禁止产品形态、标签区和配件数量漂移。
```

## 常见失败与修复

- 产品像概念设计而非真实商品：恢复准确轮廓、包装和材质；
- 背景过度华丽：降低背景饱和度、对比和细节；
- 食品看起来不可食用：恢复真实颜色、局部油光、蒸汽和纹理；
- 标签被光效遮挡：移动高光和动态元素；
- 产品悬浮像贴图：补阴影、反射和空间深度；
- 系列图风格漂移：固定角度、占比、背景、主光和文字安全区。
