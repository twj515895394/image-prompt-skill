# 3D 产品可视化风格

## 解决什么问题

定义写实 CGI 产品渲染、概念产品视觉、爆炸图、剖面图和收藏品展示的实现方式，适用于真实摄影难以完成的结构展示和理想化商业视觉。

## 什么时候读取

- 用户明确要求 3D 产品渲染、CGI、爆炸图、透明剖面或产品结构展示；
- 手办、模型、科技产品和包装需要理想化材质与灯光；
- 产品需要悬浮、拆解、内部部件或不可能机位；
- 需要区分实拍产品摄影和建模渲染。

## 核心公式

```text
准确建模
+ 清楚结构层级
+ 可信材质参数
+ 受控棚拍灯光
+ 稳定相机和比例
+ 商业级渲染完成度
```

## 主要方向

### 写实 CGI 单品

- 产品轮廓和尺寸关系准确；
- 倒角、接缝、按钮、折边和开口清楚；
- 材质粗糙度、反射和透明度可信；
- 背景简洁，渲染接近高端产品摄影；
- 不增加不存在的结构。

### 爆炸图

- 部件沿单一主轴线展开；
- 层级顺序清楚；
- 部件数量受控；
- 每个部件保持真实比例和朝向；
- 主体整体轮廓仍可被理解；
- 标注和文字排版属于 graphic design，不在本页展开。

### 透明剖面与内部结构

- 外壳透明范围明确；
- 内部部件不穿透；
- 透明材质不遮蔽关键结构；
- 保持外形识别；
- 内部结构必须逻辑可信，不能随机堆零件。

### 手办与收藏品

- 人物本体、底座、支撑杆、夹具和包装分别描述；
- 明确支撑件数量；
- 树脂、PVC、金属和透明件材质区分；
- 雕刻、衣褶和涂装清楚；
- 支撑结构不能穿透身体或抢主体。

### 概念产品主视觉

- 可以使用悬浮、几何空间和理想化背景；
- 仍需保持产品形态和品牌主色；
- 特效只强化功能或定位；
- 不能因未来感而随机改造产品结构。

## 材质控制

| 材质 | 3D 表现重点 |
|---|---|
| 金属 | 受控反射、微粗糙度、倒角高光 |
| 玻璃 | 折射、厚度、边缘和内部透视 |
| 塑料 | 表面粗糙度、颜色和注塑感 |
| 橡胶 | 低反射、柔软边缘和细微纹理 |
| 织物 | 纤维、褶皱、厚度和缝线 |
| 树脂 / PVC | 细腻涂装、适度高光和雕刻细节 |
| 纸盒包装 | 折边、印刷面、纸张厚度和轻粗糙度 |

## 相机与构图

- 产品结构展示优先 3/4 视角或正交感视角；
- 爆炸图避免强广角，防止部件比例失真；
- 手办展示可以使用轻低角度，但不改变角色比例；
- 悬浮产品需要阴影、反射或空气层次；
- 系列图固定相机高度、焦段感、背景和主光。

## Prompt 结构

```text
高完成度 3D 产品可视化，准确建模 [产品]，
固定 [轮廓、比例、结构和配件数量]，
采用 [写实 CGI / 爆炸图 / 透明剖面 / 手办展示]，
从 [视角] 展示 [关键结构]，
材质为 [材质及粗糙度、反射或透明特征]，
使用 [棚拍灯光和背景]，
保留可信接触、阴影、反射和部件层级，
禁止随机新增结构、穿透、比例漂移和材质混淆。
```

## 常见失败与修复

- 3D 产品像玩具：提高材质准确性、倒角和尺度细节；
- 爆炸图部件乱飞：固定单一轴线、顺序和间距；
- 透明剖面看不清：降低透明范围并提高内部结构分离；
- 手办支撑件错误：明确数量、位置和连接点；
- 金属表面全白：增加粗糙度和受控环境反射；
- 概念视觉改了产品结构：重复禁止变化项并减少特效干扰。
