# 3D Rendering Styles

当任务需要 3D 动画、写实 CGI、游戏过场、黏土、玩具、产品渲染、等距微缩世界或材质展示时读取本分类。

## 3D 产品可视化

当用户要求写实 CGI 产品图、爆炸图、透明剖面、手办展示、悬浮结构或真实摄影难以完成的产品视觉时，读取：

- `product-visualization.md`

## 等距微缩世界与剖面场景

当用户要求等距城市、微缩世界、建筑剖面、模拟经营场景、基地总览、地形模型或多层功能空间，并需要统一尺度、功能分区和连接结构时，读取：

- `isometric-miniature-world.md`

## 边界

- 本目录保存建模感、材质、灯光、渲染、微缩模型和三维空间表现方式；
- 具体产品、角色、场景和游戏资产交付类型进入 libraries；
- 产品展示类型进入 `../../libraries/object-product/`；
- 游戏资产交付类型进入 `../../libraries/composition-shot/game-asset-presentation-types.md`；
- 产品 Hero 构图和棚拍灯光进入对应 controls；
- 真实商业产品摄影进入 `../photography/product-photography.md`；
- 纯 2D 像素艺术进入 `../illustration/pixel-art-visual-language.md`。
