# Anime Styles

当任务需要日式动画、动画电影、赛璐璐、厚涂动画或其他动画化视觉时读取本分类。

## 电影级动画主视觉

当用户要求单幅动画 Key Visual、角色登场、游戏宣传图、宏大世界观动漫插画或高完成度动画电影感画面，并需要稳定角色、英雄姿态、环境尺度和受控特效时，读取：

- `cinematic-anime-key-visual.md`

## 边界

- 本目录保存线条、上色、造型、光影和动画质感的实现方式；
- 漫画分格、对白和拟声词进入 `../comic-manhwa/`；
- 真实摄影电影剧照进入 `../cinematic/`；
- 具体角色类型、服装、场景和游戏资产交付类型进入 libraries；
- 角色资产定义优先读取 `../../tasks/character-assets/playbook.md`。
