# 电影级动画主视觉

## 解决什么问题

定义适用于角色登场、游戏宣传、动画封面和高完成度单帧的动画主视觉语言，使角色、环境、动作和特效形成统一画面，而不是普通动漫人物站姿或漫画分格。

## 什么时候读取

- 用户要求日式动画电影感、动画 Key Visual、游戏角色宣传图或高完成度动漫插画；
- 角色需要与宏大城市、战场、自然环境或世界观同框；
- 画面需要英雄姿态、低角度、强尺度和动态特效；
- 用户要求单幅动画主视觉，而不是漫画页或技术型角色资产板。

## 核心公式

```text
稳定角色身份
+ 一个明确动作或英雄姿态
+ 强前中后景
+ 可读环境尺度
+ 统一线稿与动画渲染
+ 受控特效和光色
```

## 角色表现

- 角色脸型、发型、服装和身体比例优先稳定；
- 全身主视觉需要清楚脚部、重心和轮廓；
- 姿态应表达身份、能力或剧情状态；
- 手势、衣摆、头发和配件沿同一动作方向；
- 服装细节可以丰富，但不能破坏大轮廓；
- 强透视下仍需保护脸部和标志性服装结构。

## 构图

### 英雄低角度

适合角色登场、力量和世界观展示：

- 角色占据前景主要面积；
- 地平线较低；
- 建筑、载具或天空在后方建立尺度；
- 避免广角导致头小腿长或脚部畸变。

### 环境叙事型

适合旅行、情绪和动画电影单帧：

- 人物与环境共同表达主题；
- 环境有明确地标、天气和时间；
- 人物仍是第一视觉中心；
- 不需要漫画分格和对白框。

### 动作高潮型

适合战斗、运动和技能释放：

- 一个清楚动作阶段；
- 特效沿动作路径组织；
- 保留接触点、目标和结果方向；
- 不把多个技能高潮堆在同一张图。

## 动画渲染

可选择：

- 清洁线稿 + 赛璐璐阴影；
- 精细线稿 + 柔和动画电影上色；
- 2D / 3D 混合动画渲染；
- 厚涂动画概念插画。

同一画面固定：

- 线条粗细范围；
- 阴影边缘软硬；
- 高光形状；
- 材质简化程度；
- 脸部与头发的渲染方式；
- 环境与角色的细节等级。

## 环境与尺度

- 使用建筑、车辆、路灯、小人物、飞行器或自然地貌提供尺度参照；
- 环境细节必须支持世界观和角色，不做随机机械堆叠；
- 前景可使用地面反射、碎片、植物或结构框景；
- 中景负责动作和人物关系；
- 远景负责世界尺度和气氛；
- 大场景仍应保持角色轮廓和脸部可读。

## 光影与特效

- 发光线路、霓虹、雨水、粒子、火花和全息元素只选择必要类型；
- 特效不遮挡脸、手、武器接触点和服装标志；
- 地面反射与主光方向一致；
- 角色边缘光服务主体分离，不做全身均匀发光；
- 镜头光晕和体积光数量受控；
- 主色、辅助色和强调色形成清楚层级。

## 与相邻风格的边界

### 与漫画 / 韩漫

- 动画主视觉是单幅完整画面；
- 漫画强调分格、对白、拟声词和阅读节奏；
- 单幅动画图不默认加载漫画版式和多格连续性。

### 与电影剧照

- 动画主视觉允许更理想化姿态、环境和光效；
- 电影剧照强调可被镜头记录的剧情瞬间；
- 用户明确要真实摄影或真人电影感时，不读取本页。

### 与角色资产板

- Key Visual 优先宣传冲击和世界观；
- 角色资产板优先三视图、比例、表情和细节；
- 定义角色身份时先做资产，再做强主视觉。

## Prompt 结构

```text
电影级动画主视觉，角色为 [身份和外观锚点]，
正处于 [单一动作或英雄姿态]，
采用 [景别、机位与构图类型]，
前景包含 [框景或反射]，中景突出角色，远景用 [建筑 / 地貌 / 载具] 建立尺度，
使用统一的 [线稿与动画上色方式]，
主光来自 [合理光源]，特效沿 [动作方向] 组织，
保持脸部、服装和关键道具清楚，禁止漫画分格、过量光效和角色身份漂移。
```

## 常见失败与修复

- 只有人物站姿：加入身份相关动作、环境尺度和明确摄影位置；
- 角色被城市和特效淹没：提高主体占比并减少背景亮点；
- 画面像漫画页：移除分格、气泡和拟声词；
- 强透视导致变形：降低广角感并保护身体比例；
- 全身都在发光：只保留边缘光和功能性发光区域；
- 角色与背景像两张图：统一光向、色温、阴影和空气透视。
