# Illustration Styles

当任务需要商业插画、绘本、水彩、油画、素描、国画、像素艺术或其他绘画媒介时读取本分类。

## 像素艺术

当用户要求像素画、复古游戏画面、Sprite、像素场景、像素图标或像素战斗画面，并需要统一逻辑分辨率、有限色板、像素簇和无抗锯齿边缘时，读取：

- `pixel-art-visual-language.md`

## 边界

- 细分媒介先作为叶子 Reference；只有资料量和使用频率足够时再升级为独立子目录；
- 体素和低多边形 3D 进入 `../3d-rendering/`；
- 动画主视觉进入 `../anime/`；
- 水彩、油画、素描和水墨仍保留为审计候选，不因分类名称存在就建立空泛叶子。
