# 连续漫画视觉语言

## 解决什么问题

定义黑白漫画、彩色漫画、韩漫和条漫如何通过线稿、上色、阴影、速度线、格框和对白区域形成可读的连续叙事，而不是多张插画拼接。

## 什么时候读取

- 用户明确要求漫画、韩漫、日漫页、条漫或美式漫画；
- 需要黑白网点、彩色赛璐璐、速度线、拟声词和格框；
- 多格画面风格漂移；
- 对白框、旁白框和拟声词需要进入画面；
- 需要区分成品漫画与制作型故事板。

## 核心公式

```text
统一角色和场景
+ 可读分格
+ 连续动作与反应
+ 一致线稿和上色
+ 对白与拟声词安全区
+ 格间节奏
```

## 媒介方向

### 黑白日漫

- 清洁或有力度的黑色线稿；
- 黑白块面和网点层次；
- 集中线、速度线和冲击线；
- 背景可按情绪简化；
- 高潮格允许更强黑白反差；
- 不把灰度照片滤镜当作漫画网点。

### 彩色韩漫 / 条漫

- 清楚轮廓与柔和或赛璐璐上色；
- 肤色、发色和服装色跨格稳定；
- 背景可简化，但空间锚点需周期性出现；
- 竖向滚动通过留白、长格和近景控制节奏；
- 情绪特写可以放大眼神、手部和嘴角，但不能变脸。

### 美式动作漫画

- 更粗重的轮廓和结构阴影；
- 强透视、动态姿态和冲击构图；
- 高饱和或高对比色块；
- 拟声词和动作线与运动方向一致；
- 肌肉和动作可以夸张，但骨架与身份保持稳定。

### 轻喜剧 / 四格

- 表情和动作简洁明确；
- 起承转合清楚；
- 背景服务笑点，不必每格复杂；
- 最后一格承担反转或结果；
- 不要把四格做成四张无关联头像。

## 线稿与上色

跨格固定：

- 线条粗细范围；
- 脸部和头发轮廓规则；
- 阴影形状；
- 上色方式；
- 高光强度；
- 背景简化程度；
- 格框和纸面质感。

动作高潮可以增加线条力度，但不能突然切换另一种渲染方式。

## 对白、旁白与拟声词

- 对白框靠近说话者，尾巴指向明确；
- 旁白框与对白框在形状或位置上区分；
- 拟声词沿动作、冲击或环境声音方向放置；
- 文字不遮挡五官、手部、产品标签和关键动作；
- 阅读顺序与分格顺序一致；
- 精确长对白优先后期排版或局部编辑；
- 生成阶段可使用短对白占位和明确安全区。

## 背景与场景

- 第一格或场景切换时建立空间；
- 连续对话中可以简化背景，但需保留门窗、桌椅、道路或光向等锚点；
- 情绪格可使用抽象背景、网点或色块，但下一格应能回到真实空间；
- 不能因背景简化而改变人物左右关系和道具位置。

## Prompt 结构

```text
采用 [黑白日漫 / 彩色韩漫 / 美式漫画 / 四格] 视觉语言，
统一 [线稿、上色、阴影和背景简化方式]，
角色的脸型、发型、服装和配色跨格稳定；
每格展示一个明确事件、动作阶段或反应，
使用 [速度线 / 集中线 / 网点 / 色块] 强化 [具体节奏]，
为对白、旁白和拟声词预留清楚区域，
格间形成连续叙事，而不是独立插画拼贴。
```

## 与故事板的边界

- 成品漫画服务读者阅读和最终视觉；
- 故事板服务镜头设计和后续视频制作；
- 漫画页可以有对白框、拟声词、不规则分格和出血画面；
- 故事板通常保留镜号、时段、动作和技术标注；
- 用户只要求制作分镜时，不默认加载漫画视觉语言。

## 常见失败与修复

- 每格画风不同：固定线稿、阴影、上色和背景规则；
- 对白框不知道谁在说话：明确尾巴方向和说话者位置；
- 速度线到处使用：只在真实动作或冲击时使用；
- 背景全部消失：定期恢复空间锚点；
- 情绪特写导致变脸：重复脸部稳定区并控制夸张强度；
- 漫画页像故事板草稿：增加成品线稿、上色、格框和阅读节奏；
- 故事板被过度漫画化：移除对白装饰和不必要的成品特效。
