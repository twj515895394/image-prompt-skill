# Styles Index

风格层负责说明某种视觉风格如何实现，不负责罗列该风格下所有人物、服装和场景选项。

只有用户明确指定风格，或风格会决定线条、渲染、材质和光影实现时，才读取 0–1 份风格 Reference。

- 摄影与照片质感：`photography/index.md`
- 电影画面与影视视觉：`cinematic/index.md`
- 日本动画及相关动画风格：`anime/index.md`
- 漫画、韩漫和条漫表现：`comic-manhwa/index.md`
- 插画、绘本和传统绘画媒介：`illustration/index.md`
- 角色、场景和工业概念设计：`concept-art/index.md`
- 3D 动画、写实渲染和材质表现：`3d-rendering/index.md`
- 海报、版式、品牌与平面视觉：`graphic-design/index.md`

风格资料只回答“如何呈现”。具体人物、服装、场景和道具选择进入 libraries。
