# 产品棚拍灯光控制

## 解决什么问题

根据产品材质、轮廓和商业目标选择主光、轮廓光、反射和阴影方式，避免所有产品都使用同一种“柔和棚拍光”。

## 什么时候读取

- 电商主图、产品广告、食品广告或产品系列图；
- 金属、玻璃、塑料、树脂、织物、食品等材质需要准确表现；
- 画面出现反光失控、轮廓不清、标签过曝或暗部死黑；
- 产品与背景无法分离；
- 系列图需要统一光向与阴影。

## 控制顺序

```text
材质与产品结构
→ 主光方向和面积
→ 辅光与暗部
→ 轮廓光
→ 反射控制
→ 接触阴影
→ 背景亮度与色温
```

## 材质与灯光

| 材质 | 灯光重点 | 避免 |
|---|---|---|
| 金属 | 长条高光、边缘清楚、受控镜面反射 | 高光断裂、全白反光、表面像塑料 |
| 玻璃 | 边缘亮线、透射、折射和背景分离 | 轮廓消失、内部浑浊、反射杂乱 |
| 塑料 / 树脂 | 柔和高光、适度粗糙度、颜色准确 | 过度油亮、蜡质或廉价感 |
| 陶瓷 | 连续柔光、釉面高光和体积阴影 | 高光太硬、杯口或边缘变形 |
| 织物 | 斜侧光、纤维、褶皱和接触阴影 | 光线太平导致材质不可读 |
| 皮革 | 柔硬结合、纹理和边缘高光 | 表面像塑料、黑色无层次 |
| 食品 | 湿润度、油光、蒸汽、切面和酥脆纹理 | 全局油亮、颜色不自然、食物像塑料 |
| 包装纸盒 | 平整受光、折边和印刷区域清楚 | 标签过曝、纸面像金属 |

## 常用灯光结构

### 干净电商光

- 大面积柔和主光；
- 弱填充光恢复暗部；
- 轻接触阴影；
- 中性背景；
- 标签和结构优先可读。

### 奢侈品戏剧光

- 方向明确的侧光或顶侧光；
- 局部轮廓光；
- 深色背景与受控高光；
- 暗部仍保留材质纹理。

### 透明材质边缘光

- 背后或侧后方大面积亮源；
- 前方弱填充；
- 通过背景明暗变化描出边缘；
- 避免正面闪光覆盖标签。

### 食品广告光

- 侧后方主光突出纹理和蒸汽；
- 前侧弱反射光保护食物颜色；
- 配料、液滴和切面有局部高光；
- 保持可食用质感。

### 动态饮料光

- 罐体或瓶体有清楚轮廓；
- 冰块、果片和液滴接受同一方向光；
- 霓虹或彩色轮廓光只作为产品主色辅助；
- 深色背景不能吞掉产品边缘。

## 阴影和接触

- 落地产品需要明确接触阴影；
- 悬浮产品需要软阴影、反射或空气层次说明高度；
- 台座产品的阴影方向必须与主光一致；
- 多产品组合的阴影不能互相穿透；
- 食品和配料需要真实接触，不应漂浮。

## 标签与品牌色保护

- 标签区域避免强高光和折射；
- 品牌主色不能被环境色完全污染；
- 黑色产品需要边缘分离，白色产品需要背景明度差；
- 金属包装的高光应避开关键信息区；
- 文字准确性不由灯光页负责，但灯光不能破坏可读性。

## Prompt 结构

```text
根据 [材质] 设计棚拍光，
主光从 [方向] 以 [软硬与面积] 照亮 [关键结构]，
辅光保留 [暗部细节]，
轮廓光用于分离 [边缘]，
高光集中在 [材质关键区域]，避开 [标签或 Logo]，
产品与 [地面 / 台座 / 悬浮空间] 形成可信阴影或反射，
背景亮度和色温不改变产品真实颜色。
```

## 常见失败与修复

- 金属像塑料：增加连续条状反射和真实粗糙度；
- 玻璃消失：用背景明暗和边缘光描出轮廓；
- 标签过曝：移动高光并降低正面反射；
- 黑色产品无细节：增加边缘亮度和暗部层次；
- 食品像塑料：降低统一高光，增加局部湿润与真实纹理；
- 悬浮无重量：补软阴影、反射和明确高度；
- 系列图光线漂移：固定主光方向、产品角度和阴影落点。
