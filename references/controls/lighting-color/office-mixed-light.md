# 办公室混合光控制

## 用途

建立办公室、茶水间、会议室和走廊中的普通顶灯、窗光与环境反射关系，避免把职场纪实画面打成棚拍或广告光。

## 何时读取

- 职场场景需要真实工作日光线
- 人物和环境光色不一致
- 办公室画面过于柔美、完美或像商业摄影

## 常见光源结构

### 顶灯主导

- 冷白或中性白顶灯承担基础照明
- 阴影不会像棚灯一样极度柔和
- 桌面、墙面和人物脸部存在轻微亮度差

### 窗边自然光补充

- 靠窗区域获得较柔和的侧光或环境光
- 不要求形成戏剧化轮廓光
- 与顶灯共同造成轻微色温差

### 局部环境反射

- 显示器、玻璃隔断、白色墙面和金属边框会产生冷白反射
- 地面、桌面或电梯门可能形成局部高光
- 反射应服务空间材质，不应变成霓虹效果

## 控制顺序

```text
主光来源
→ 窗光或环境补光
→ 阴影硬度
→ 色温差
→ 材质反射
→ 曝光与高光限制
```

## Prompt 写法

```text
普通办公室顶灯作为基础照明，窗边自然光从侧面轻微补入，人物与环境共享同一光线逻辑，冷白环境反射落在玻璃隔断和桌面，阴影边缘自然但不刻意柔化，保留真实工作空间中的轻微色温差和明暗不均
```

## 禁止倾向

- 完美柔光棚拍
- 过强轮廓光
- 大面积金橙电影光
- 人物亮而背景完全不受光
- 假 HDR 与过曝高光

## 常见失败与修复

- 人物像单独打光：让皮肤、衣服和背景共享顶灯与窗光
- 光线太平：保留普通空间中的区域亮度差
- 光线太电影化：减少轮廓光和色彩对比，恢复工作日照明
- 冷白光像医院：加入窗光、木质或肤色反射保持自然层次
