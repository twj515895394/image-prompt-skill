# 电影光影控制模式

## 解决什么问题

把“电影光、氛围光、高级光影”等空泛描述，拆成可执行的光源方向、阴影关系、色温组织和主体分离方式，使镜头、空间、材质和情绪能够互相支持。

## 什么时候读取

- 用户明确要求电影感、戏剧性照明或类型片气氛；
- 画面需要通过光线表现人物权力、脆弱、悬疑、浪漫或仪式感；
- Prompt 只有色调词，没有光源方向和阴影结构；
- 人物或产品无法从背景中分离；
- 多个光型互相冲突，或高光、阴影和材质表现失控。

普通办公室现场光优先读取 `office-mixed-light.md`；只有需要电影化光影判断时再读取本页。

## 控制顺序

```text
场景中的合理光源
→ 主光方向
→ 辅光与环境光
→ 阴影软硬和密度
→ 高光位置
→ 主体分离方式
→ 色温与情绪
```

先建立光源动机，再添加氛围效果。

## 常用光型

| 光型 | 可见结构 | 适合场景 | 风险 |
|---|---|---|---|
| 平光 | 面部和物体受光均匀、阴影弱 | 电商、信息清晰、轻松日常 | 缺少立体感和情绪 |
| 侧光 | 一侧受光、一侧进入阴影 | 人物肖像、冲突、悬疑 | 暗侧细节丢失 |
| 逆光 | 背景光源形成边缘亮线和空气感 | 回忆、浪漫、离别、登场 | 面部变黑、背景过曝 |
| 顶光 | 光从上方压下，眼窝和下颌阴影明显 | 审讯、压迫、舞台、权力空间 | 人脸难看、暗部死黑 |
| 底光 | 光从下方照亮，五官阴影反常 | 惊悚、怪诞、地下空间 | 容易变成无目的恐怖效果 |
| 轮廓光 | 主体边缘被勾亮 | 登场、广告、舞台、暗背景 | 轮廓光过强像后期描边 |
| 伦勃朗光 | 面部形成稳定三角亮区 | 成熟肖像、严肃角色、人物海报 | 不适合要求轻快自然的场景 |
| 明暗对比光 | 亮区和暗区边界明确 | 黑色电影、动作、危机、权谋 | 整体曝光过低 |
| 丁达尔光 | 可见光束穿过雾、尘或空气颗粒 | 教堂、森林、废墟、仪式空间 | 光束过多抢主体 |

## 主体分离方法

优先从以下方式中选择一到两种：

- 亮暗分离：主体比背景更亮或更暗；
- 色温分离：主体暖、背景冷，或主体冷、背景暖；
- 轮廓分离：边缘光勾出人物或产品外轮廓；
- 饱和度分离：主体保留关键色，背景降低色彩强度；
- 清晰度分离：主体更清楚，背景适度柔化；
- 空间层分离：前景、中景和背景分别接受不同强度的光。

不要同时把所有分离方式写满。

## 阴影控制

- 软阴影：适合生活化、浪漫、人像和商业柔光；
- 硬阴影：适合正午、悬疑、权力、动作和设计感画面；
- 阴影必须保留必要纹理，除非题材明确需要剪影；
- 人像优先保护眼神、肤色和面部识别区；
- 产品优先保护轮廓、材质反射和品牌颜色；
- 场景优先保护空间层次和出入口方向。

## 色温关系

电影光影通常由关系而非单一颜色成立：

```text
阴影色 + 高光色 + 主体分离
```

常用关系：

- 冷阴影 + 暖高光：都市、商业、夜景、动作；
- 暖环境光 + 中性肤色：家庭、回忆、室内关系；
- 冷环境光 + 局部暖光：孤独、悬疑、夜行人物；
- 中性光 + 低饱和阴影：纪实、克制现实；
- 深色背景 + 金色轮廓：奢侈品、权力、登场。

具体色调方案读取 `../../libraries/lighting-color/cinematic-color-palettes.md`。

## 与镜头协调

- 近景和特写：优先保护面部、眼神或产品关键材质；
- 广角环境镜头：光线需要说明空间层级，不只照亮主体；
- 仰拍：避免顶光进一步加重眼窝死黑；
- 俯拍：注意头顶高光和脸部受光是否失衡；
- 逆光镜头：补可解释的面部反射光或环境辅光；
- 前景遮挡镜头：前景不能亮到抢走第一视觉落点。

## 组合模板

```text
场景主要由 [合理光源] 照明，
主光从 [方向] 落在 [主体关键区域]，
[软 / 硬] 阴影建立 [情绪或空间关系]，
辅光控制 [暗部细节]，
通过 [亮暗 / 色温 / 轮廓] 让主体从背景分离，
高光集中在 [材质或轮廓位置]，避免全局泛光。
```

## 常见失败与修复

- 只有“电影光”没有来源：补现实可解释的主光和环境光；
- 高光铺满全图：把高光限制在轮廓、皮肤或材质关键区；
- 逆光导致主体全黑：补弱面部反射或调整背景曝光；
- 暗部死黑：降低对比强度，恢复纹理；
- 多种光型互相打架：保留一个主光逻辑，其他光只做辅助；
- 光影与镜头冲突：重新检查相机方向、主体朝向和光源位置；
- 光线漂亮但场景不可信：让窗户、灯具、霓虹、火焰或天空成为可见光源依据。
