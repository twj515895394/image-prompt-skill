# 生活化微表情控制

## 用途

当用户要求“自然一点”“像活人”“别像面瘫”“像朋友或同事随手拍到”时，把情绪落实为视线、眉眼、嘴角、呼吸和身体小动作。

## 何时读取

- 人物近景、特写和半身成片；
- 图生图需要把表情改得更自然；
- 角色资产中的头像和表情板；
- 视频参考帧中的情绪首帧或生活化特写；
- 需要描述情绪如何从一个状态过渡到另一个状态。

## 核心规则

生活化人物的活人感通常不是来自夸张表情，而是：

- 眼神有明确落点；
- 眉眼发生很小变化；
- 嘴角状态克制；
- 呼吸或回答前有短暂停顿；
- 肩颈、下巴、手指参与情绪；
- 情绪像正在发生，而不是已经演完。

## 先判断

1. 当前是不是强情绪；
2. 这个瞬间处于起始、发生中还是结果态；
3. 人物正在看什么，或刻意避开什么；
4. 哪一个局部变化是主锚点；
5. 身体是否有与情绪一致的小动作。

## 常用方向

### 日常亲和

- 刚被逗笑，笑意还没完全收住；
- 想回应对方，嘴角先动了一点；
- 轻微害羞，视线先移开再回来。

### 克制疲惫

- 礼貌微笑，但眼底略有疲惫；
- 假装镇定，回答前停顿半拍；
- 苦笑一下，很快收住。

### 关系拉扯

- 欲言又止，嘴唇轻轻收紧；
- 听到熟悉话题，眼神先停一下；
- 轻微吃醋，视线移开但表情没有完全冷掉；
- 心疼但不敢靠近，身体重心略向前又停住。

### 识破与警觉

- 听到破绽后，目光短暂停住；
- 眉间略收，嘴角维持原状；
- 下巴轻轻收回，身体先出现防备。

## 单图终态与视频过程

### 单图或静态参考帧

只描述当前可见终态：

```text
眼神落点 + 眉眼状态 + 嘴角状态 + 呼吸感 + 小动作
```

不要把多个时间阶段同时塞进一张图。

### 视频首帧、尾帧或表演过程

需要时可拆成两个短阶段：

```text
0–1.5s：眼神、呼吸和嘴角的起始状态
1.5–3s：眉头、泪水、唇角、下颌或手部的变化
```

每个阶段只保留一个主要变化，避免脸部所有区域同时动作。

## Prompt 写法

```text
自然生活化微表情，人物没有完全看向镜头，
视线落在 [对话对象 / 手中动作 / 某个信息来源]，
眉眼 [轻微变化]，嘴角 [克制变化]，
呼吸 [短暂停顿 / 缓慢呼出]，
肩颈、下巴或手指保留 [小动作]，
情绪处于 [开始 / 发生中 / 收尾]，
而不是准备好给镜头看的标准表情。
```

## 负向约束

- no exaggerated expression
- no frozen smile
- no doll-like blank stare
- no overacting face
- no forced cute pose
- no melodramatic emotion

负向词只作为补充，不能代替正向可见变化。

## 常见失败与修复

- 只写微笑：补眼神落点和嘴角变化；
- 情绪词太大：改为一个正在发生的小瞬间；
- 表情细但身体僵硬：增加肩颈、手指或下巴的小动作；
- 一直直视镜头：让视线服务当前关系或动作目标；
- 静态图包含多个时间阶段：只保留当前终态；
- 视频变化过多导致变脸：每个阶段只控制一个主变化，并保持脸型和五官比例稳定。
