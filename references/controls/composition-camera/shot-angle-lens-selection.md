# 景别、机位与焦段选择控制

## 解决什么问题

把“远景、特写、俯拍、仰拍、24mm、85mm、浅景深”等镜头词，转成基于任务目标、主体关系和空间表达的选择方法，避免互斥镜头参数堆叠。

## 什么时候读取

- 用户只给出粗粒度镜头要求，需要补全景别、机位或焦段；
- 画面重点依赖主体与环境比例；
- 人物关系、压迫感、脆弱感或临场感需要由观看位置建立；
- Prompt 中出现广角与长焦、深景深与强虚化等冲突；
- 分镜、角色资产、场景资产或成片任务需要明确镜头入口。

普通办公室随手抓拍优先读取 `office-candid-camera.md`；只有需要通用镜头判断时再读取本页。

## 选择顺序

```text
画面任务
→ 主体与环境谁更重要
→ 观看距离
→ 观看高度和方向
→ 透视关系
→ 焦段感
→ 景深范围
```

不要先选摄影器材品牌，再反推画面。

## 景别选择

| 景别 | 主要作用 | 适合任务 | 常见风险 |
|---|---|---|---|
| 远景 | 交代世界、位置和尺度 | 开场、旅行、城市、自然、灾难、世界观 | 主体过小、情绪不可读 |
| 全景 | 交代全身、动作和空间关系 | 站位、动作、群像、角色资产 | 背景过多抢主体 |
| 中景 | 建立人物关系和事件 | 对话、互动、冲突前奏、生活场景 | 既无空间信息又无情绪重点 |
| 近景 | 推进情绪或手部、道具细节 | 人物反应、关系拉扯、关键动作 | 裁切关节或丢失环境依据 |
| 特写 | 锁定证据、材质或情绪爆点 | 眼神、嘴角、伤口、产品细节、关键道具 | 缺少上下文，容易像孤立素材 |

判断原则：

- 需要说明“在哪里、人物如何分布”时，优先远景、全景或中景；
- 需要说明“人物正在感受什么”时，优先近景或特写；
- 同一张图只保留一个主景别，不写“超远景特写”这类互斥组合；
- 可以用“中景为主，保留手部和部分环境”表达过渡，而不是堆两个景别名。

## 机位选择

| 机位 | 视觉关系 | 适合场景 | 控制重点 |
|---|---|---|---|
| 平视 | 自然、平等、可信 | 日常、对话、写真、产品正面展示 | 相机高度与主体视线一致 |
| 俯拍 | 脆弱、渺小、空间清楚 | 受挫、孤立、桌面、站位规划 | 避免脸部和身体被过度压缩 |
| 仰拍 | 强势、压迫、英雄感 | 登场、觉醒、权力人物、巨构 | 控制腿部和下颌透视变形 |
| 鸟瞰 | 总览布局、强调孤立 | 城市、战场、群像、场景规划 | 主体识别度与空间方向 |
| 荷兰角 | 不稳定、失衡、心理异常 | 悬疑、危机、崩塌 | 只在叙事需要时使用 |
| POV | 主观临场、代入 | 追逐、惊吓、互动、第一人称 | 明确观看者位置和可见肢体 |
| 过肩 | 建立关系、对话和窥视 | 双人戏、谈判、冲突 | 前景肩部不能遮挡关键表情 |
| 前景遮挡 | 增加在场感和空间层次 | 街拍、观察、纪实、悬疑 | 遮挡应合理，不要误盖主体 |

## 焦段与透视

| 焦段感 | 空间表现 | 常见用途 | 风险 |
|---|---|---|---|
| 24mm 广角 | 空间展开、近大远小明显 | 环境、建筑、动作、近距离压迫 | 面部、四肢和产品边缘变形 |
| 35mm 纪实广角 | 保留环境且接近人在场观察 | 街拍、生活流、短剧、办公室 | 主体太靠边时视觉重心不稳 |
| 50mm 标准视角 | 自然、稳定、关系清楚 | 人像、对话、通用成片 | 画面可能过于平常，需要构图和事件支撑 |
| 85mm 中长焦 | 背景压缩、主体突出 | 肖像、情绪特写、商业人像 | 环境信息不足，容易变棚拍 |
| 微距 | 放大细节和材质 | 食物、花瓣、机械、皮肤、产品结构 | 景深过浅、整体形态不可读 |
| 鱼眼 | 强边缘扭曲和潮流冲击 | 滑板、街头、演唱会、实验视觉 | 不适合要求准确比例的任务 |

原则：

- 焦段决定空间关系，不只是“画质标签”；
- 需要环境和动作时优先 24–35mm；
- 需要自然人物关系时优先 35–50mm；
- 需要肖像压缩和背景分离时优先 85mm；
- 产品形态必须准确时，谨慎使用极广角和鱼眼；
- 单镜头通常只写一个主要焦段感。

## 景深选择

- 浅景深：用于人物、产品和情绪焦点，但必须保留足够识别信息；
- 中等景深：兼顾主体与环境，适合对话、短剧、商业成片；
- 深景深：适合群像、场景资产、建筑、动作和空间规划；
- 不同时要求“全景深清晰”和“背景完全化开”；
- 故事板和场景资产优先可读性，不默认使用极浅景深。

## 组合模板

```text
[景别]，[机位与观看方向]，
使用 [焦段感] 建立 [空间或人物关系]，
[浅 / 中等 / 深] 景深，
主体占画面 [比例与位置]，
保留 [必要环境 / 动作 / 情绪细节]。
```

## 常见失败与修复

- 广角同时要求长焦压缩：回到主体与环境比例，只保留一种主要透视；
- 景别很多但主次不明：明确第一视觉落点并保留一个主景别；
- 俯拍或仰拍导致身体变形：降低角度强度，增加合理拍摄距离；
- 人像背景虚化过度：恢复可识别环境证据；
- 场景资产过度电影化：优先空间可读和稳定镜头入口；
- 只写相机品牌：改为焦段、透视、景深、色彩和材质等可见结果。
