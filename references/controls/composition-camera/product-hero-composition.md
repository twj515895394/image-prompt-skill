# 产品 Hero 构图控制

## 解决什么问题

决定产品在商业画面中的主体占比、拍摄角度、视觉动线、留白和辅助元素关系，确保第一眼先看见产品，而不是背景、特效或文案。

## 什么时候读取

- 电商主图、产品广告、品牌主视觉和产品系列图；
- 需要单品居中、低角度、3/4 视角、对角动态、悬浮或组合陈列；
- 产品形态正确，但画面缺乏主次或商业冲击力；
- 动态元素、台座或场景抢主体；
- 需要为 Logo、标题、卖点或价格区预留空间。

## 控制顺序

```text
销售目标
→ 第一视觉落点
→ 产品角度与主体占比
→ 重心和接触关系
→ 辅助元素动线
→ 背景层级
→ 文案安全区
```

## 常用 Hero 模式

| 模式 | 视觉作用 | 适用产品 | 控制重点 |
|---|---|---|---|
| 正面居中 | 直接、准确、稳定 | 包装、瓶罐、电子产品 | 对称、标签可读、轻阴影 |
| 3/4 角度 | 同时展示正面和侧面体积 | 杯具、箱包、美妆、家电 | 不变形，保留关键结构 |
| 轻俯拍 | 展示顶部、开口和组合关系 | 食品、杯具、礼盒、平铺 | 避免产品被压扁 |
| 低角度仰拍 | 强势、奢华、纪念碑感 | 美妆、饮料、科技、汽车部件 | 控制透视和底部结构 |
| 对角线动态 | 能量、速度和年轻感 | 饮料、运动、潮流商品 | 产品仍为最大主体 |
| 悬浮中心 | 轻盈、科技、未来感 | 耳机、手表、护肤品、包装 | 软阴影或反射提供空间依据 |
| 台座主视觉 | 稳定、精品、陈列感 | 珠宝、美妆、奢侈品 | 台座材质和产品接触 |
| 主次组合 | 展示套装和系列 | 礼盒、护肤套装、多口味 | 主产品更大、更亮或更靠前 |
| 包装与内容物 | 同时解释品牌和实际内容 | 食品、玩具、收藏品 | 包装、实物和配件不混淆 |

## 主体占比

- 电商单品主图：产品通常占画面较高比例，避免环境喧宾夺主；
- 品牌广告：允许更多负空间，但产品仍需保持第一视觉权重；
- 组合陈列：主产品通过尺寸、位置、明度或清晰度领先；
- 食品广告：主体食物应完整可读，配料只做口味和质感证据；
- 爆炸图：整体结构居中，部件沿清楚轴线展开。

不要用固定百分比代替判断，但应在 Prompt 中明确“主体占画面大部分 / 中等比例 / 保留大面积文案留白”。

## 辅助元素动线

液体、果片、冰块、花瓣、碎屑、火花和零件应：

- 沿一条主方向组织；
- 指向或围绕产品；
- 避开标签、Logo 和关键结构；
- 数量受控；
- 远近大小符合透视；
- 不形成第二个视觉中心。

## 留白与文字安全区

- 预先指定标题、Logo、卖点或价格区域；
- 产品轮廓不要侵入全部留白；
- 背景在文案区应更平整、对比稳定；
- 文字区不放高频反光、碎片和复杂纹理；
- 具体排版与文字层级留给 Phase 6 的 graphic design Reference。

## 系列图一致性

产品系列图应固定：

- 相机高度和焦段感；
- 产品占比；
- 背景结构；
- 主光方向；
- 阴影位置；
- 文案安全区；
- 允许变化的口味色、配料和角度范围。

## Prompt 结构

```text
以 [产品] 为第一视觉中心，采用 [Hero 模式]，
从 [视角] 展示 [关键结构]，主体占画面 [比例描述]，
产品重心稳定并与 [台座 / 地面 / 悬浮阴影] 建立可信关系，
辅助元素沿 [方向] 围绕主体，但不遮挡 [标签 / 轮廓 / 功能部位]，
在 [位置] 保留干净文案安全区，
背景和特效不得形成第二视觉中心。
```

## 常见失败与修复

- 产品太小：提升主体占比并降低背景复杂度；
- 特效抢主体：减少数量、饱和度和亮度；
- 低角度导致瓶体或箱包变形：降低广角感并增加拍摄距离；
- 套装平均排布：确定一个主产品和明确层级；
- 悬浮像贴纸：补软阴影、反射或空气透视；
- 文案区不可用：重新建立平整负空间并移除复杂元素。
