# Quick Mode

## 目标

在尽量不打扰用户的前提下，直接给出稳妥可用、能复制即用的生图 Prompt。

## 适用信号

- 用户明显要结果，不要过程
- 用户描述简短，但方向已足够明确
- 当前不确定项主要是镜头、光影、材质这类可稳补变量

## 默认动作

1. 先锁当前任务的主目标。
2. 用输入信息补主体、场景、镜头、光影、风格、限制六层。
3. 给 `1` 条主 Prompt。
4. 只有当第二条真的有价值时，再给 `1` 条备选 Prompt。

## 允许主动补全

- 景别、机位、焦段
- 光源方向
- 阴影色和高光色
- 真实感材质
- 与任务相关的负向限制

## 不允许主动补全

- 用户没有暗示的题材切换
- 强剧情关系
- 世界观设定
- 大幅改写人物身份
- 把写实改成二次元，或反过来

## 什么时候必须打断追问

- 连主体是谁都不清楚
- 多张参考图互相冲突且无法判定主图
- 用户要求明显互斥
- 角色资产任务缺少固定锚点，结果一定会飘

## 输出骨架

```text
画面概述：
主要约束：
最终 Prompt：
备选 Prompt：
已自动补全项：
```

## 自查清单

- 是否先写主体，再写风格
- 是否把“电影感”拆成镜头、光影、色调和材质
- 是否给了真正和任务相关的限制项
- 是否避免用一堆漂亮空词代替执行变量
