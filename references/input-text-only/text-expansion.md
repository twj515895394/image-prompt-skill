# Text Expansion

## 目标

把一句话创作意图扩成一套可执行画面结构，而不是只补几个漂亮形容词。

## 通用扩写顺序

1. 主体
2. 场景
3. 动作或状态
4. 镜头
5. 光影
6. 色调
7. 风格与材质
8. 比例与限制

## 先补哪类变量

### 第一优先级：不补就会跑偏

- 主体身份
- 场景地点
- 当前任务目标

### 第二优先级：影响画面质感

- 景别
- 机位
- 光线方向
- 色调结构

### 第三优先级：影响完成度

- 材质真实感
- 局部细节
- 负向限制

## 按任务调整扩写重点

### 成片生图

优先补：

- 主体气质
- 镜头与光影
- 服装与材质
- 情绪氛围

### 角色资产

优先补：

- 固定身份
- 固定外貌锚点
- 固定服装配色
- 表情与姿态组

### 场景资产

优先补：

- 空间层级
- 站位与动线
- 主道具
- 镜头入口

## 不主动补全

- 未被暗示的强剧情
- 明显改变题材的大方向
- 用户没有要求的世界观设定
- 会把单图任务变成长篇叙事的内容

## 输出草稿骨架

```text

场景：
动作/状态：
镜头：
光影：
色调：
风格与材质：
比例：

```
