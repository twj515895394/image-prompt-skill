# Scene Assets Playbook

## 目标

生成可复用的场景资产 Prompt，让封面、分镜和图生视频都能共享同一空间锚点。

## 核心结构

1. 地点类型
2. 时间与天气
3. 空间层级
4. 关键道具
5. 人物站位
6. 动线与视线
7. 镜头入口
8. 光影色调
9. 负向约束

## 空间卡必须回答

- 这是哪里
- 主体从哪儿进
- 观众第一眼看哪里
- 哪个道具是主道具
- 这个空间未来如何承接动作或镜头运动

## 资产清单维度

### 空间结构层

- 前景遮挡物
- 中景主体区
- 背景世界信息
- 空间纵深
- 出入口位置

### 叙事调度层

- 人物站位
- 对峙方向
- 进入与离开动线
- 视线落点
- 道具落点

### 美术锚点层

- 主道具
- 副道具
- 场景材质
- 主色调
- 氛围光源

### 视频衔接层

- 可运动元素
- 镜头推进入口
- 横移入口
- 环绕入口
- 可剪辑节点

## 推荐输出包

1. 场景总览图
2. 站位图
3. 空景图
4. 道具关系图
5. 镜头入口图

## 输出骨架

```text
场景用途：
空间结构：
站位与动线：
主道具 / 副道具：
镜头入口：
场景资产 Prompt：
限制项：
```

## 常见失败

- 只有环境描写，没有空间层次
- 道具很多，但没有主道具层级
- 场景好看，却无法承接人物动作和镜头运动

## 修复方式

- 空间扁平：补前中后景和出入口
- 空镜漂亮但没法拍：补站位、动线和镜头入口
- 道具乱：明确主道具和视觉落点
