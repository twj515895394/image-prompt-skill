# Composition & Shot Library

当任务需要具体景别、机位、构图模板、主体占比、多人构图或镜头方案库时读取本分类。

## 漫画分格与页面版式

当用户需要黑白漫画、彩色漫画、韩漫、条漫、四格或多格叙事，并需要根据阅读方向、动作节奏、对白量和高潮位置选择分格时，读取：

- `comic-panel-layouts.md`

## 海报与封面版式

当海报、活动主视觉、社交媒体封面、Banner 或 YouTube 缩略图需要根据主体、文字量、平台比例和阅读距离选择版式时，读取：

- `poster-layout-types.md`

## 故事板类型与版式

当用户需要故事板总板、控制型故事板、风格型故事板、动作拆解板、场景调度板或关键帧板，并需要根据镜头数量选择布局和信息密度时，读取：

- `storyboard-board-types.md`

## 边界

- 站位、动线和跨镜方向由 `../../controls/spatial-blocking/` 负责；

- 漫画视觉语言进入 `../../styles/comic-manhwa/`；
