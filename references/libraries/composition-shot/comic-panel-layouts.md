# 漫画分格与页面版式选择器

## 解决什么问题

根据叙事节奏、阅读方向、动作强度和信息密度选择漫画分格，而不是把所有漫画都排成规则九宫格。

## 什么时候读取

- 黑白漫画、彩色漫画、韩漫、条漫、四格和多格叙事；
- 需要决定规则格、不规则格、大格、小反应格、出血格或滚动留白；
- 对话、动作和高潮需要不同节奏；
- 用户要求“漫画页”而不是制作型故事板。

## 页面类型

| 类型 | 适用场景 | 节奏特点 |
|---|---|---|
| 规则网格 | 对话、说明、稳定推进 | 阅读清楚、节奏均匀 |
| 大格 + 小反应格 | 高潮、揭示、强情绪 | 主事件突出，反应补充 |
| 横向宽格 | 环境、群像、对峙 | 空间清楚、节奏放缓 |
| 纵向长格 | 下落、奔跑、楼梯、纵深 | 强方向感和持续运动 |
| 窄格连续切片 | 快速动作、眼神、手部细节 | 节奏加速、时间切碎 |
| 对角或不规则格 | 战斗、冲突、失衡 | 动势强，但阅读风险高 |
| 无边框 / 出血格 | 记忆、梦境、高潮、冲击 | 突破页面，强调情绪 |
| 四格 | 短笑点、反转、步骤 | 起承转合明确 |
| 竖向条漫 | 手机阅读、情绪停顿、揭示 | 通过滚动距离控制节奏 |
| 跨页大图 | 世界观、终极高潮、重要登场 | 视觉震撼，信息应简洁 |

## 选择顺序

```text
阅读媒介
→ 阅读方向
→ 场景数量
→ 叙事节拍
→ 高潮位置
→ 对白量
→ 分格类型
```

## 节奏控制

- 大格延长停留并强化重要性；
- 小格加快阅读或补充反应；
- 连续窄格适合动作细节和微表情变化；
- 留白可以制造停顿、时间流逝和情绪距离；
- 高潮前可使用规则格积累，高潮时用大格或出血格释放；
- 同一页不要使用过多不同形状的分格。

## 对话页

- 每格通常只承担一个主要说话行为或反应；
- 对白区需要提前预留；
- 说话者位置和阅读方向一致；
- 反应格可以无对白，但应推进关系；
- 对话多时优先规则格或双栏，不使用过度复杂斜格。

## 动作页

- 先建立空间和人物位置；
- 动作按准备、发力、接触、结果拆分；
- 大格用于关键打击、跃迁或转折；
- 速度线和拟声词沿动作方向组织；
- 下一格起点应承接上一格动作终点。

## 条漫

- 通过垂直间距控制停顿；
- 重要揭示前增加空白；
- 对话不要跨越过长滚动距离；
- 同一场景的背景锚点需间隔出现，避免失去空间感；
- 手机屏幕一次可见内容应有明确焦点。

## Prompt 结构

```text
生成一页 [漫画类型]，采用 [阅读方向]，
共 [格数] 格，使用 [版式类型]。
第 [格] 为高潮大格，其余格按 [节奏] 推进；
每格只承担一个主要事件、动作阶段或反应；
为对白和拟声词预留清楚区域，
保持阅读顺序明确，不使用过多互相冲突的分格形状。
```

## 常见失败与修复

- 所有格同样大：根据高潮和反应重新分配面积；
- 阅读顺序混乱：减少斜格，明确左到右、右到左或上到下；
- 对白遮挡人物：提前建立对白安全区；
- 动作页没有空间：先用一格交代人物和场景位置；
- 条漫过长但无节奏：用留白、背景锚点和揭示位置分段；
- 每格都是独立插画：重写输入、输出和承接关系。
