# Libraries Index

资料库负责提供具体选项、选择器和详细参考，不负责讲解完整控制方法。

按用户需要的具体内容读取 0–2 份资料。

- 人物、妆容、发型、服装、表情与体态资料：`human/index.md`
- 室内、室外、自然、城市和功能空间资料：`environment/index.md`
- 产品、道具、交通工具和物件资料：`object-product/index.md`
- 动物、幻想生物和非人角色资料：`animal-creature/index.md`
- 构图、景别和镜头方案库：`composition-shot/index.md`
- 光型、色板和配色方案库：`lighting-color/index.md`
- 材质、表面、粒子和视觉效果资料：`material-effect/index.md`

新增资料先分析其自身用途、结构和主要检索轴，再决定整份保留或拆分。
