# Environment Library

当任务需要具体室内、室外、自然、城市、办公、商业或功能空间资料时读取本分类。

## 办公与职场空间

当场景是办公室、茶水间、工位区、会议室、电梯口或走廊，需要具体环境锚点、道具密度和空间用途时，读取：

- `office-workplace-environment.md`

## 边界

- 空间站位、动线和镜头方法归入 `controls/`
- 光线判断归入 `controls/lighting-color/`
- 摄影表现归入 `styles/photography/`

后续家庭、街道、商业、工业、自然和幻想环境资料按实际需要补充。
