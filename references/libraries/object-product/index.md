# Object & Product Library

当任务需要具体产品、道具、包装、器具、家具、交通工具或物件资料时读取本分类。

## 产品展示类型

当电商主图、商业广告或产品系列图需要选择白底、极简、台座、悬浮、动态、场景化、组合陈列、包装与内容物、爆炸图、剖面或手办展示时，读取：

- `product-display-types.md`

## 边界

- 本目录保存可选择的对象类型、结构、材质和展示资料；
- Hero 构图和主体占比进入 `../../controls/composition-camera/`；
- 棚拍光和材质照明进入 `../../controls/lighting-color/`；
- 产品摄影与 3D 可视化进入对应 `styles/`；
- 对象替换、空间绑定和编辑约束仍由任务与 controls 负责。
