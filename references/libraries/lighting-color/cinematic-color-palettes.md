# 电影色调选择器

## 解决什么问题

提供可直接选择的电影色调方案，并说明适用题材、核心关系和常见风险。它回答“有哪些色调可以选”，不代替光源方向、阴影和主体分离的控制判断。

## 什么时候读取

- 用户明确要求某种电影色调或类型片色彩；
- 需要为人物、场景、产品或海报选择稳定的阴影色与高光色关系；
- 用户只说“高级灰、青橙、港风、赛博”等粗粒度方向；
- 画面出现全图发灰、肤色偏色、霓虹失控或暗部死黑。

如果任务只需要判断光源和阴影，不需要选择具体色调，则读取 `../../controls/lighting-color/cinematic-lighting-patterns.md`，不必加载本页。

## 选择结构

```text
题材与情绪
→ 阴影色
→ 高光色
→ 主体保留色
→ 饱和度与对比度
→ 风险控制
```

## 常用方案

| 色调方向 | 核心关系 | 适用场景 | 常见风险 |
|---|---|---|---|
| 青橙电影感 | 青蓝阴影 + 暖橙高光 | 都市、商业、动作、夜景 | 肤色过橙、对比过强 |
| 冷蓝悬疑 | 蓝灰阴影 + 冷白高光 | 调查、走廊、雨夜、危机 | 主体过暗、画面单调 |
| 复古胶片 | 暖灰或褪色阴影 + 柔和高光 + 轻颗粒 | 回忆、年代感、旅行、室内 | 颗粒像噪点、白色发脏 |
| 低饱和高级灰 | 中性灰阶 + 主体局部保色 | 极简空间、产品、时尚 | 全图发灰、没有第一视觉落点 |
| 暖黄怀旧 | 棕灰阴影 + 金黄高光 | 家庭、旧时光、室内回忆 | 肤色和白墙发黄 |
| 黑金商业 | 深中性阴影 + 金色轮廓或高光 | 奢侈品、权力、登场、广告 | 暗部死黑、金色泛滥 |
| 暗黑犯罪 | 冷绿或蓝黑阴影 + 少量硬高光 | 黑巷、犯罪、动作、危机 | 人脸不可读、脏灰 |
| 日系清透 | 浅灰阴影 + 自然暖白高光 | 夏日、街道、生活写真 | 过曝、绿色荧光化 |
| 韩剧柔光 | 奶油中间调 + 柔暖高光 | 室内关系、浪漫人物 | 过度磨皮、缺少层次 |
| 赛博霓虹 | 蓝紫阴影 + 洋红或青色高光 | 科幻、夜城、潮流海报 | 霓虹铺满、主体被吞没 |
| 末日废土 | 灰黄或土褐阴影 + 苍白高光 | 荒原、废墟、灾后 | 全图发黄发脏 |
| 纪录片自然色 | 中性阴影 + 真实肤色和现场高光 | 街头、生活观察、纪实 | 缺少重点或被误做成滤镜 |
| 商业广告色 | 干净中性背景 + 准确产品主色 | 产品、品牌主视觉、电商 | 产品颜色漂移、反光过度 |
| 都市夜景 | 蓝灰阴影 + 暖路灯高光 + 局部霓虹 | 夜行人物、都市短剧 | 面部太暗、背景光点过多 |
| 夏日清爽 | 天空蓝、自然绿色 + 明亮暖白高光 | 海边、旅行、女友感、夏日海报 | 绿色过饱和、天空过曝 |
| 阴天情绪 | 冷灰阴影 + 柔散射高光 | 克制关系、街景、孤独 | 灰成一片、主体不分离 |
| 森林绿色 | 深浅绿色分层 + 叶间暖光 | 森林、冒险、疗愈 | 绿色合成单一色块 |
| 公路夕阳 | 蓝紫或棕灰阴影 + 金橙夕阳 | 公路片、旅行、尾镜 | 天空过曝、人物变剪影 |

## 选择规则

- 一张图通常选择一个主色调方向；
- 强风格画面中，色调只支撑主风格，不争夺主导权；
- 人像先保护肤色和眼神，产品先保护品牌色与材质，场景先保护空间层次；
- “低饱和”不是把全图统一变灰，应保留主体辨识色；
- “霓虹”应集中在少量光源和反射区，不等于全画面蓝紫洋红；
- 同一镜头不要叠加互斥方案，如青橙、奶油韩剧和废土灰黄同时作为主色调。

## Prompt 组合模板

```text
整体采用 [色调方向]，
阴影呈 [阴影色与饱和度]，
高光呈 [高光色与强度]，
主体保留 [肤色 / 产品主色 / 关键服装色]，
通过 [亮暗 / 冷暖 / 饱和度] 与背景分离，
避免 [该方案的主要风险]。
```

## 常见失败与修复

- 青橙导致肤色过橙：降低高光橙色强度，恢复中性肤色；
- 高级灰没有重点：为主体保留一种识别色或亮度优势；
- 暖黄画面发脏：让白色保持中性，暖色只作用于光源和局部反射；
- 黑金暗部无纹理：抬升阴影并保留材质细节；
- 赛博霓虹失控：限制光源数量和色彩区域；
- 胶片颗粒像噪点：降低颗粒强度，保留主体清晰度；
- 阴天画面灰平：增加前中后景明度差和局部主体分离。
