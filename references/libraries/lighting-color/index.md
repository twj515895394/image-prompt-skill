# Lighting & Color Library

当任务需要具体光型、色板、冷暖方案、时间天气光线或题材配色资料时读取本分类。

## 电影色调方案

当用户需要从青橙、冷蓝悬疑、复古胶片、低饱和高级灰、黑金、赛博霓虹、纪录片自然色、夏日清爽等方案中选择，并需要了解适用题材和风险时，读取：

- `cinematic-color-palettes.md`

## 边界

- 本目录回答“有哪些光线和配色可以选”；
- 如何协调光源、阴影、空间、镜头和材质由 `../../controls/lighting-color/` 负责；
- 用户只需要光源判断时，不额外加载色调选择器；
- 本索引不重复叶子中的具体色调列表。
