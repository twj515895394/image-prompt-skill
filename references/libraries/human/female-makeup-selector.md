# 女主妆容选择器

## 定位

用于女主成片、角色定妆图、AI 漫剧封面和人物写真。妆容是角色气质开关，应与发型、人物外貌、光影色调、服装和场景同步选择。

## 何时读取

- 妆容直接决定人设气质
- 用户只说“淡妆、浓妆、清冷、御姐、病娇”等粗粒度方向
- 角色资产需要建立稳定妆容锚点
- 成片或图生图需要把妆容与题材、职业和光影协同

## 妆容 Prompt 结构

```text
人设气质 + 底妆质感 + 眉眼 + 眼线/眼影 + 腮红/唇色 + 光影色调 + 服装场景
```

## 协同规则

- 清纯系妆容优先搭配柔和光线、浅色服装和近距离镜头
- 强势系妆容优先搭配方向明确的侧光、深色服装和中近景
- 破碎系妆容优先搭配低饱和色调、局部乱发和情绪镜头
- 暗黑系妆容优先搭配反差光影和更强背景控制
- 职业角色优先干净、克制和可信赖，避免无必要舞台妆

## 避坑

- 妆容不能只写“好看”，应说明底妆、眼妆、唇色和角色气质
- 甜妹妆和暗黑女王妆不要混用，除非明确要反差人物
- 病娇和战损妆需要控制苍白、泪痕和红眼尾强度
- 不要让所有人设都变成同一种精致网红妆
- 妆容、发型、服装和光影不能分别选择后直接拼接，应服务同一人物目标
