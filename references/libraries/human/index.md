# Human Library

当任务需要具体人物外观、妆容、发型、服装、表情、体态或角色类型资料时读取本分类。

## 女主妆容

当妆容决定人设，或需要根据白月光、甜妹、学姐、名媛、御姐、病娇、反派、古风和题材角色选择妆容时，读取：

- `female-makeup-selector.md`

## 角色发型

当发型是身份锚点，或用户只提供黑长直、短发、卷发等粗粒度词时，读取：

- `hairstyle-selector.md`

## 角色表情

当角色需要表情板、情绪终态或具体表情方向时，读取：

- `expression-selector.md`

## 通勤服装与面料

当用户要求通勤感、职场人物、知性现代角色，或需要通过面料和版型表现自然体态线条时，读取：

- `commuter-outfit-materials.md`

## 边界

- 姿态、重心和动作控制归入 `controls/pose-action/`
- 微表情如何自然发生归入 `controls/pose-action/lifestyle-micro-expression.md`
- 本目录只保存可选择的具体资料和选择器
