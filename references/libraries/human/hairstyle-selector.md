# 角色发型选择器

## 定位

把“黑长直”“短发”“卷发”等粗粒度描述，细化为可用于角色一致性和 Prompt 组装的发型锚点。

## 何时读取

- 发型是角色辨识度的重要部分
- 用户只给出粗粒度发型词，需要细化
- 角色资产需要跨角度、跨表情和跨场景保持一致
- 妆容、职业或题材需要与发型协同

## 选择顺序

```text
角色身份与气质
→ 基础长度
→ 发型形态
→ 刘海或分线
→ 发色
→ 配饰
→ 发丝质感和动态
```

## 按角色气质选择

### 校园与清爽

- 黑长直
- 高马尾
- 半扎发
- 齐肩发

适合清纯、青春、初恋、校花和邻家角色。

### 职场与干练

- 低马尾
- 鲨鱼夹盘发
- 短发 BOB
- 低盘发

适合律师、医生、设计师、管理者和现代职场人物。

### 御姐与强势

- 黑长直
- 大波浪
- 法式卷
- 侧分长发

适合权力感、成熟气场、港风和财阀角色。

### 病娇与反差

- 双马尾
- 姬发式
- 水母头
- 麻花辫

适合脆弱与危险并存、黑莲花和病娇人物。

### 甜妹与元气

- 高马尾
- 丸子头
- 蛋卷头
- 丝带辫

适合偶像、甜妹、校园和青春题材。

## 发型锚点字段

- 长度
- 形态
- 刘海或分线
- 发色
- 配饰
- 发丝质感
- 动态状态

角色一致性任务中至少固定前五项。

## 镜头与动态联动

- 长发、大波浪和云朵卷适合风吹、回头和运动画面
- 高马尾、双马尾适合动作感与青春感
- 低马尾、短发 BOB 适合冷静、专业和职业角色
- 湿发、凌乱发丝适合雨夜、战损和破碎状态
- 近景和侧颜画面需要明确发丝是否遮挡脸部轮廓

## 组合模板

```text
[基础长度]，[发型形态]，
[刘海或分线]，[发色]，[配饰]，
发丝呈现 [质感与动态]，
整体服务 [角色身份与气质]。
```

## 避坑

- 只写黑长直，不写分线、刘海、长度和发丝状态
- 每张图都随机改变刘海或分线
- 动作画面中发丝完全静止
- 发型与职业、时代和人物气质冲突
- 发型动态遮住关键五官，却没有明确遮挡范围
