# Image Reference Analysis

## 目标

把参考图拆成可执行变量，用来服务当前目标，而不是照抄整张图。

## 先判参考图类型

- 成片图
- 角色图
- 场景图
- 分镜图
- 首帧 / 尾帧 / 关键帧

不同类型决定保留项和改写项，不要一套分析方式打天下。

## 主图与辅图规则

### 主参考图负责

- 主体身份
- 核心气质
- 主场景锚点
- 主要构图关系

### 辅助图负责

- 妆容
- 发型
- 服装
- 光影
- 镜头
- 局部风格借用

不做平均融合。

## 分析顺序

1. 主体是谁，观众先看哪里
2. 这是成片图还是资产图
3. 镜头距离、角度、焦段是什么
4. 光源从哪里来，阴影边缘是硬还是软
5. 色调、材质、服装、妆容各自承担什么作用
6. 当前任务里哪些要保，哪些要改

## 保留 / 改写矩阵

### 默认优先保留

- 主体身份
- 核心气质
- 关键材质

### 默认允许改写

- 局部构图细枝末节
- 背景杂项
- 纯装饰元素
- 与当前任务无关的氛围细节

## 冲突处理

当文字要求与图片内容冲突时，优先级如下：

1. 用户当前文字目标
2. 用户特别强调必须保留的图片锚点
3. 主参考图
4. 辅助图

## 输出草稿骨架

```text
参考图类型：
主图保留项：
辅助图借用项：
可改写项：
镜头要点：
光影要点：
材质要点：
当前任务限制：
```
