# Image To Image Playbook

## 目标

在保留关键锚点的前提下完成定向改写，而不是把原图复述一遍。

## 先判改写强度

### 复刻

- 主体、气质、场景、构图都尽量接近
- 只做轻微质感优化或错误修复

### 轻再创作

- 保主体和气质
- 可改镜头、光影、服装、妆容、色调

### 强改写

- 只借主体锚点或局部风格
- 新目标优先级高于原图完整保留

## 组装顺序

1. 识别主参考图
2. 列保留项
3. 列替换项
4. 列只借用不复刻的风格项
5. 补当前目标的镜头、光影、限制

## 保留项优先级

- 主体身份
- 核心气质
- 主场景锚点
- 关键材质

## 常见替换项

- 妆容方向
- 发型状态
- 服装类型
- 色调
- 光影方式
- 构图重心
- 情绪强度

## 什么时候继续下钻控制页

如果图生图目标不只是“换个风格”，而是明确追求下面这些结果，不要只停在本页：

- `看不出是 AI`
- `像真实照片`
- `像手机抓拍`
- `办公室 / 茶水间 / 工位纪实`
- `漂亮侧颜`
- `自然体态`
- `别太工整`
- `像顺手拍到`

优先下钻：

- `style-control/anti-ai-realism.md`
- `style-control/failure-patterns-negative.md`
- `style-control/candid-composition-imperfections.md`
- `portrait/side-profile-bodyline.md`
- `scene-office/office-candid-photography.md`

特别注意：

- 原图已经有抓拍基础时，优先保留构图和站位证据，再补真实感控制
- 原图像棚拍或网红写真时，先补失败模式约束，再决定是否继续保留原图光影
- 如果改写目标是“更像真实职场侧拍”这类明确方向，控制页优先级高于空泛风格词

## 修复方式

- 结果太像原图：减少构图与背景细节硬保留
- 结果漂移太大：补主体锚点和关键材质
- 只有风格变化，没有画面变化：补镜头、光影和构图改写目标
