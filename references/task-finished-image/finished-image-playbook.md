# Finished Image Playbook

## 目标

输出一张高完成度结果图的 Prompt，而不是资产清单或灵感摘要。

## 最少需要锁定的东西

- 主体是谁
- 画面要表达什么情绪或事件
- 场景落在哪里
- 镜头离主体多近

如果这四件事里缺两件以上，先别硬写长 Prompt。

## Prompt 组装顺序

1. 主体身份与气质
2. 场景与空间关系
3. 镜头与构图
4. 光影与色调
5. 服装、妆容、材质
6. 氛围与细节
7. 任务相关限制

## 主体层怎么写

- 先写身份，再写气质
- 写清脸型、体态、眼神时，优先服务主体辨识度
- 不要让风格词压过主体词
- 如果侧颜和体态是卖点，先保证自然比例和姿态成立，再考虑“高级感”

## 镜头层怎么写

- 至少给出景别或机位中的一个
- 电影感必须拆成镜头、光线、空间层次
- 如果用户没给明确镜头，优先补最能服务情绪的那一种，而不是最花哨的那一种

## 光影层怎么写

- 先定光源方向
- 再定阴影气质
- 再定色调结构

不要只写“高级电影光影”，要写清是逆光、侧光、冷蓝阴影还是暖橙高光。

## 真实感怎么保

- 真实感来自具体视觉线索，不来自反复堆 `photorealistic`
- 抓拍感来自构图、站位、光线与空间缺陷，不来自“candid”单词本身
- 如果场景是办公室，保留真实道具密度和普通灯光痕迹，比堆“高级办公空间”更稳

## 输出骨架

```text
画面概述：
主要约束：
最终 Prompt：
[主体]，[场景]，[动作/状态]，
[镜头与构图]，[光影与色调]，
[服装、妆容、材质]，[氛围与细节]，
[比例与限制]

备选 Prompt：
```

## 常见失败

- 主体不清楚，风格词过重
- 场景漂亮但没有情绪指向
- 妆容、服装、光影互相打架
- 只有空泛质感词，没有具体镜头和光源
- 画面像棚拍写真，不像生活中真的会拍到的瞬间
- 侧颜和身材被写成模板化网红图，而不是自然体态

## 修复方式

- 主体模糊：减风格词，增身份和脸部锚点
- 画面脏乱：收窄场景道具与背景信息
- 妆容跑偏：单独补妆容方向和服装协同
- 电影感空心：补镜头距离、光源方向、阴影与色调
- 抓拍失真：补拍摄者站位、构图缺陷、环境道具与普通光线
- 人体失真：补肩颈线、重心、腰臀关系与动作阶段
