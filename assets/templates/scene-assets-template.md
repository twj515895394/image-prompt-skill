# 场景资产交付模板

仅在用户要求完整、结构化交付时读取。

```markdown
# 场景资产 Prompt

## 场景目标

[说明场景用途及后续服务对象。]

## 空间固定项

- 地点与用途：
- 时间、天气与状态：
- 空间分区：
- 出入口与动线：
- 主道具与副道具：
- 光影与色彩：

## 人物与镜头关系

- 人物站位：
- 视线关系：
- 镜头入口：
- 可变区域：

## 主 Prompt

[完整场景资产 Prompt。]

## 推荐输出顺序

1. 场景总览
2. 空景
3. 站位或动线图
4. 多角度或道具细节

## 一致性限制

[空间、道具、出入口、光线和比例稳定项。]
```
