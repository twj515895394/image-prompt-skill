# 视频参考帧交付模板

仅在用户要求完整、结构化交付时读取。

```markdown
# 视频参考帧 Prompt

## 帧任务

[首帧 / 尾帧 / 起止双帧 / 关键过渡帧]

## 跨帧固定项

- 人物身份与服装：
- 场景结构：
- 机位与画幅：
- 主道具：
- 光线与风格：

## 可运动项

- 肢体与重心：
- 头发与衣摆：
- 道具运动：
- 环境动态：
- 镜头允许变化：

## 首帧 Prompt

[动作起点与运动空间。]

## 尾帧 Prompt

[动作结果与终态。]

## 关键过渡帧 Prompt

[只有首尾无法稳定表达时提供。]

## 起止关系与禁止突变

[动作方向、允许变化和跨帧禁止项。]
```
