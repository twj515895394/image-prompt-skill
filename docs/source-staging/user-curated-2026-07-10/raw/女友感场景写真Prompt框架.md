# 女友感场景写真 Prompt 框架

## 定位

用于把单条案例 Prompt 抽象成可复用框架，覆盖水果、旅行、自拍、胶片抓拍、漂浮感等女友感场景。它连接 [[人物外貌与写真Prompt结构]]、[[电影光影与色调选择器]] 和 [[女友感写真]]。

## 核心判断

女友感不是固定题材，而是“亲近视角 + 生活事件 + 轻微不完美 + 真实光线 + 随手构图”的组合。

## 框架

```text
亲近关系视角 + 日常场景 + 轻动作 + 可触摸道具 + 真实光线 + 轻微不完美 + 平台画幅
```

## 变量表

| 维度 | 可选方向 | 写法重点 |
|---|---|---|
| 关系视角 | 男友视角、朋友视角、前置自拍、随手抓拍 | 让镜头像人在场 |
| 日常场景 | 野餐、窗边、果园、海峡旅行、散场、泻湖 | 场景要能承载动作 |
| 轻动作 | 回头、递水果、整理头发、漂浮、走路、自拍 | 动作不要像摆拍大片 |
| 道具 | 草莓、青提、手机、包、餐布、湿发、饮料 | 道具要参与画面关系 |
| 光线 | 自然光、傍晚光、窗边光、胶片闪光 | 避免棚拍感过强 |
| 不完美 | 轻微虚焦、发丝凌乱、衣角褶皱、皮肤纹理 | 增强真实感 |
| 画幅 | 3:4、4:5、9:16 | 适配小红书、短视频封面或手机壁纸 |

## 可复用模板

```text
[关系视角] 下拍到 [人物] 在 [日常场景] 中 [轻动作]，
画面包含 [道具与人物互动]，
[真实光线]，[轻微不完美细节]，
[镜头/焦段/景深]，[色调]，[画幅比例]。
```

## 不沉淀原文 Prompt 的原因

- 草莓野餐、青提、果园、海峡旅行、散场自拍、泻湖漂浮属于同类案例。
- 长期库只保留可迁移变量，避免以后检索时被大量单条案例淹没。
