# 表情与微表情选择器

## 定位

用于 AI 漫剧角色表演、表情表、情绪特写和图生视频首帧设计。它连接 [[角色资产生成结构]]、[[导演级运镜术语选择器]] 和 [[打击帧与动作戏结构]]。

## 两层结构

| 层级 | 用途 | 写法重点 |
|---|---|---|
| 表情 | 明确情绪类型 | 嘴角、牙齿、眼神、眉峰、面部肌肉、光影 |
| 微表情 | 让情绪更真实 | 眼神落点、眉眼变化、嘴角变化、呼吸停顿、小动作 |

## 表情 Prompt 公式

```text
表情类型 + 嘴角/牙齿 + 眼神 + 眉峰 + 面部肌肉 + 光影/色调
```

## 微表情 Prompt 公式

```text
微表情 + 眼神落点 + 眉眼变化 + 嘴角变化 + 呼吸停顿 + 小动作 + 适用场景
```

## 进阶写法

- 先定场景标签，再写微表情结构，例如欲言又止、识破谎言、病弱强撑、师徒克制、妖族蛊惑。
- 超近景和图生视频首帧可直接写时间分段：

```text
0-1.5s：眼神 / 呼吸 / 嘴角的起始状态
1.5-3s：泪水、眉头、唇角或下颌的变化
```

- 眼神通常是主锚点，嘴角和呼吸决定节奏，小动作负责活人感。

## 表情类型选择

| 类型 | 画面描述重点 | 适用场景 |
|---|---|---|
| 元气微笑 | 嘴角上扬、苹果肌饱满、眼尾弯起 | 轻松日常、校园、初见 |
| 大笑 | 露齿、眼角细纹、头微仰 | 胜利、放松、喜剧 |
| 偷笑 | 嘴角抵起、眼神狡黠、目光闪躲 | 暧昧、恶作剧、秘密 |
| 害羞笑 | 低头、脸颊泛红、眼神低垂 | 心动、被夸、暧昧 |
| 傲娇冷笑 | 单侧嘴角、眉峰微挑、下颌微抬 | 反派、御姐、对峙 |
| 苦笑 | 僵硬弧度、眉头微蹙、眼底疲惫 | 失败、告别、无奈 |
| 假笑 | 嘴角上扬但眼神空洞 | 社交伪装、权谋、隐藏情绪 |

## 微表情分组

| 分组 | 代表情绪 | 写法重点 |
|---|---|---|
| 情绪压抑 | 强忍委屈、假装镇定、秘密被戳破、忽然心动、强装冷漠 | 眼尾泛红、嘴角压住、呼吸停顿、视线避开 |
| 关系拉扯 | 欲言又止、听到坏消息、识破谎言、心疼不敢靠近、轻微吃醋 | 视线漂移、唇部收紧、手指小动作 |
| 反派权谋 | 邪魅试探、礼貌疏离、压迫审视、黑化前一秒、伪装无辜 | 眉眼控制、嘴角轻动、目光停留 |
| 古风仙侠 | 白月光回眸、病弱强撑、魔尊冷情、仙门威压、记忆苏醒 | 眼神、呼吸、衣袖和发丝配合 |
| 告别回忆 | 忍痛告别、死里逃生、默默守护、旧爱相见、听见熟悉声音 | 停顿、回望、眼眶变化、手部收紧 |
| 活人感反应 | 眉头紧锁、阴沉脸、咬牙切齿、冷眼相视、难以置信、心虚躲闪 | 少用“大怒/开心”，多写局部肌肉和视线 |

## 用力不过猛规则

- 把“开心”改成“眼睛弯起、嘴角上扬”。
- 把“愤怒”改成“眉头压低、嘴角绷紧、鼻翼轻张”。
- 把“害怕”改成“瞳孔放大、肩膀后缩、呼吸停住”。
- 把“委屈”改成“眼尾泛红、嘴角向下压、视线移开”。
- 把“冷漠”改成“眼皮半垂、嘴角拉平、回答前停顿半拍”。

## 镜头搭配

- 情绪爆点：特写、大特写、推近压迫镜头。
- 关系拉扯：中近景、过肩镜头、反打对峙镜头。
- 隐忍和告别：后撤疏离镜头、长焦抽离镜头、空镜余韵镜头。
- 秘密揭晓：拉焦转移镜头、侧移窥视镜头、慢摇预兆镜头。
- 超近景悲伤戏：固定镜头或极慢推近，允许写 0-1.5s / 1.5-3s 的表演节拍。

## 避坑

- 不要每个镜头都使用强情绪，角色会像表情包。
- 表情变化会导致五官漂移，重要角色要提前生成表情表。
- 微表情必须绑定剧情场景，否则只是细节堆叠。
- 表情、妆容、光影要一致，例如破碎感不适合高饱和甜妹光。
