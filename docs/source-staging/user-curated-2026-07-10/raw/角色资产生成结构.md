# 角色资产生成结构

## 定位

用于角色设定图、角色说明书、定妆图、表情表和后续视频一致性控制。它连接 [[人物外貌与写真Prompt结构]]、[[角色发型选择器]]、[[表情与微表情选择器]]、[[AI漫剧]] 和 [[图生视频Prompt结构]]。

## 角色说明书结构

| 模块 | 内容 | 作用 |
|---|---|---|
| 基础身份 | 姓名、年龄段、职业、关系、时代背景 | 稳定角色定位 |
| 外貌锚点 | 脸型、五官、发型、肤色、身形 | 防止变脸 |
| 服装锚点 | 主服装、材质、配饰、颜色 | 保持连续镜头一致 |
| 气质锚点 | 性格、情绪底色、眼神状态 | 控制表演方向 |
| 表情范围 | 常态、微笑、愤怒、紧张、克制 | 服务剧情表演 |
| 动作习惯 | 站姿、手势、走路方式、小动作 | 增加辨识度 |
| 禁止项 | 不要改变发色、脸型、年龄感等 | 降低漂移 |

## 稳定流程

1. 先定角色设定，写清身份、气质、服装和关键词。
2. 生成三视图，先锁正面、侧面、背面。
3. 补多角度头像，优先做 4-6 张。
4. 补表情组，先做 6 张稳定表情，再扩展到 9-12 张。
5. 补细节资产，再进入分镜、封面和图生视频。

## 固定锚点

- 发际线
- 眼距
- 鼻梁高度
- 唇形
- 下巴长度
- 耳位

尽量只改眉、眼、嘴角来控制表情，脸型和五官比例不要乱变。

## 资产类型

- 角色定妆图：正面、侧面、半身、全身。
- 表情表：常态、微笑、惊讶、压抑、愤怒、悲伤。
- 换装图：保留脸部与身形锚点，只换服装。
- 视频参考帧：保留可运动部位和镜头方向。

## 细节资产清单

- 服装结构：领口、门襟、袖口、口袋、裤型、裙摆方向。
- 材质与色板：主色、辅助色、点缀色，以及棉、针织、牛仔、皮革等材质。
- 鞋特写：正侧后视图，保证全身镜头统一。
- 手部特写：自然垂放、握物、抬手、指尖细节。
- 配饰 / 道具：耳饰、项链、包、手机、发夹、墨镜等。
- 半身主图：用于封面、角色主视觉和后续延展。

## 微表情使用原则

- 微表情应服务剧情，不要每张图都“大情绪”。
- 先定义情绪强度：克制、明显、爆发。
- 表情要和眼神、眉毛、嘴角、肩颈一起写。
- 使用 [[表情与微表情选择器]] 时，优先把情绪拆成眼神、眉眼、嘴角、呼吸和小动作。

## 发型使用原则

- 发型是角色一致性锚点，跨镜头不要频繁改变长度、刘海和发色。
- 使用 [[角色发型选择器]] 时，先按身份气质选择，再补长度、刘海、发色、配饰和动态。

## 失败模式

- 角色只有“好看”，没有稳定锚点。
- 每次换场景都换发型、脸型或年龄感。
- 表情词过猛，导致角色像夸张表情包。
- 图生视频时没有把首帧角色锚点传递到运动 Prompt。
