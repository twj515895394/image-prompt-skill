# 图像风格提取流程

## 定位

用于把参考图转成可复用 Prompt 结构，服务 [[人物外貌与写真Prompt结构]]、[[图生视频Prompt结构]] 和 [[分镜资产与image2故事版]]。

## 输入

- 一张参考图，或一组同风格参考图。
- 目标用途：生图复刻风格、角色资产、场景资产、首帧图、分镜图或视频参考帧。

## 提取维度

| 维度 | 需要观察什么 | 输出方式 |
|---|---|---|
| 色彩 | 主色、辅色、阴影色、高光色 | 写成色调组合，不只写颜色名 |
| 构图 | 主体位置、留白、前中后景 | 写成可执行构图关系 |
| 镜头 | 景别、机位、焦段、景深 | 接入 [[视角机位与景别选择器]] |
| 光影 | 光源方向、强弱、阴影边缘 | 接入 [[电影光影与色调选择器]] |
| 材质 | 皮肤、布料、金属、玻璃、胶片颗粒 | 写清质感与反光方式 |
| 场景 | 地点、道具、空间层次 | 接入 [[场景资产与空间规划]] |
| 风格 | 媒介、时代、导演气质或摄影语言 | 避免只写风格名 |
| 限制 | 参考图中不希望复现的问题 | 转成负面限制 |

## 输出模板

```text
参考图风格提取：
1. 色彩：[主色/阴影/高光/饱和度]
2. 构图：[主体位置/前景/背景/留白]
3. 镜头：[景别/机位/焦段/景深]
4. 光影：[光源方向/强弱/阴影质感]
5. 材质：[皮肤/服装/道具/环境质感]
6. 场景：[空间/道具/人物关系]
7. 风格：[媒介/时代/审美方向]
8. 限制：[不要复现的错误]
```

## 使用原则

- 先提取结构，再改写成目标题材，不要直接复制参考图内容。
- 风格复刻优先保留光影、色调、构图和材质，主体可替换。
- 如果用于视频首帧，必须额外补“可运动元素”和“镜头后续运动方向”。
