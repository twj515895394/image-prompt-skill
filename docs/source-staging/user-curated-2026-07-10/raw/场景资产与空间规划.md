# 场景资产与空间规划

## 定位

用于场景图、空间站位图、AI 漫剧场景资产和视频镜头前的空间规划。它连接 [[视角机位与景别选择器]]、[[故事版分镜到视频流程]] 和 [[场景资产与空间规划]] 自身的场景复用规则。

## 场景资产结构

```text
地点类型 + 时间天气 + 空间层级 + 关键道具 + 人物站位 + 镜头入口 + 光影色调
```

## 空间层级

| 层级 | 写什么 | 作用 |
|---|---|---|
| 前景 | 门框、窗帘、树枝、路人、桌面物 | 增加纵深和遮挡 |
| 中景 | 主体人物、主要动作区 | 承载剧情 |
| 背景 | 建筑、天光、街道、人群、远山 | 建立世界 |
| 动线 | 人物进入、离开、转身、对峙方向 | 服务视频运动 |
| 视线 | 人物看向谁或什么 | 建立关系 |

## 三角站位公式

| 站位 | 适合场景 | 效果 |
|---|---|---|
| 三角稳定 | 三人对话、团队关系 | 稳定、明确、均衡 |
| 对角线交叉 | 对峙、冲突、追逐 | 动态、紧张 |
| 放射式站位 | 众人围绕核心物或人物 | 强调中心事件 |

## 场景图到视频的额外字段

- 可运动元素：风、雨、车流、人群、窗帘、头发、水面。
- 镜头运动入口：推进、横移、环绕、跟拍、拉远。
- 可剪辑节点：人物进入、动作完成、视线转移、道具落点。
