# 电影感 Prompt 拆解公式

## 定位

不要只写“电影感”。本页把电影感拆成可控制变量，供 [[电影光影与色调选择器]]、[[视角机位与景别选择器]] 和 [[文生视频Prompt结构]] 调用。

## 核心公式

```text
电影感 = 主体清晰 + 场景可信 + 摄影风格 + 光影方式 + 色彩调性 + 景深效果 + 情绪氛围
```

## 变量表

| 变量 | 作用 | 可选方向 |
|---|---|---|
| 主体 | 明确谁在画面里 | 单人、双人、群像、物体、城市空间 |
| 场景 | 建立世界感 | 街道、房间、雨夜、海边、车内、废墟 |
| 摄影风格 | 决定画面语言 | 电影剧照、纪录片、广告片、胶片摄影、手持镜头 |
| 光影方式 | 决定立体感 | 侧光、逆光、伦勃朗光、明暗对比光、丁达尔光 |
| 色彩调性 | 决定情绪温度 | 冷青、暖橙、低饱和、高反差、复古胶片 |
| 景深效果 | 决定注意力 | 浅景深、深景深、背景虚化、前景遮挡 |
| 情绪氛围 | 决定叙事感 | 孤独、暧昧、压迫、温暖、紧张、松弛 |

## 写法示例骨架

```text
[主体] 在 [场景] 中 [动作/状态]，
采用 [摄影风格]，
[光影方式] 形成 [明暗关系]，
整体 [色彩调性]，
[景深效果] 强化视觉焦点，
氛围 [情绪关键词]。
```

## 质量判断

- 如果删掉“电影感”三个字，画面仍然能成立，说明 Prompt 有效。
- 如果只保留“电影感”三个字，画面不可控，说明结构不足。
- 光影、色调、镜头必须服务同一个情绪，不要把所有高级词都塞进去。
