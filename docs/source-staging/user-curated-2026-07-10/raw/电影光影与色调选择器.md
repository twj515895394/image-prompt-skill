# 电影光影与色调选择器

## 定位

用于给图片和视频建立质感，不再只写“高级电影色调”。它连接 [[电影感Prompt拆解公式]]、[[人物外貌与写真Prompt结构]]、[[文生视频Prompt结构]] 和 [[生图风格选择器]]。

## 光影类型

| 类型 | 视觉效果 | 情绪倾向 | 适合场景 |
|---|---|---|---|
| 平光 | 柔和、清楚、低戏剧性 | 中性、信息清晰 | 电商、干净人像、轻松日常 |
| 侧光 | 立体、轮廓明确 | 性格感、 tension | 写实人像、悬疑、冲突 |
| 逆光 | 边缘光、氛围感 | 回忆、浪漫、离别 | 夕阳、背光人物、情绪短片 |
| 顶光 | 从上压下、脸部阴影重 | 审判感、压迫感 | 审讯、神秘空间、舞台强气氛 |
| 底光 | 从下往上、五官变形 | 异常、不安、惊悚 | 惊悚、怪诞、地下空间 |
| 轮廓光 | 人物边缘被勾出 | 神圣、分离、登场 | 角色出场、舞台、广告主视觉 |
| 伦勃朗光 | 面部三角光、戏剧性 | 深沉、严肃、成熟 | 电影肖像、人物海报、严肃角色 |
| 明暗对比光 | 高反差、压迫感 | 危险、冲突、黑色感 | 黑色电影、动作、危机场景 |
| 丁达尔光 | 光束、空气颗粒、空间深度 | 神秘、诗意、仪式感 | 教堂、森林、尘埃空间、废墟 |

## 电影级色调三结构

```text
色调 = 阴影色 + 高光色 + 主体分离
```

| 结构 | 问题 | 写法 |
|---|---|---|
| 阴影色 | 暗部是什么颜色 | 冷青阴影、蓝灰阴影、墨绿色阴影 |
| 高光色 | 亮部是什么颜色 | 暖橙高光、金色夕阳、冷白霓虹 |
| 主体分离 | 主体如何从背景里出来 | 边缘光、色温对比、亮暗分离 |

## 组合规则

- 先选 1 类光影，再选 1 个色调方向，最后决定主体分离方式。
- 人像优先保证肤色和主体边缘，场景图优先保证空间层次和阴影秩序。
- 同一张图或同一镜里，不要同时堆多个互斥色调，例如“青橙 + 韩剧奶油肤色 + 废土灰黄”。
- 如果已经用了强风格的 [[影视风格选择器]]，色调层只做支撑，不要反过来抢主风格。

## 检查表

- 是否写清光源方向。
- 阴影色和高光色是否服务同一个情绪。
- 主体是否从背景中分离。
- 色调是否和服装、场景、题材一致。

## 常见失败模式

- 把色调当滤镜，不写阴影色、高光色和主体分离。
- 只有“电影感”标签，没有光源方向、对比关系和空间层次。
- 先把全图统一降饱和，再发现主体和背景一起死掉。
- 把漂亮的局部高光写成全局高光，结果画面脏、爆、乱。
