# 角色发型选择器

## 定位

本页用于角色设定、写真成片、AI 漫剧定妆图和角色一致性控制。发型不是装饰词，而是角色身份、年龄感、气质和镜头动态的锚点。它连接 [[人物外貌与写真Prompt结构]]、[[角色资产生成结构]] 和 [[AI漫剧]]。

## 发型 Prompt 结构

```text
基础长度 + 发型形态 + 刘海/分线 + 发色 + 配饰 + 质感 + 角色气质 + 场景身份
```

## 镜头动态规则

- 长发、云朵卷、大波浪适合风吹、奔跑、回头、慢镜头。
- 高马尾、双马尾、丝带辫适合动作感和青春感。
- 低马尾、低盘发、短发 BOB 适合职场对白和冷静角色。
- 湿发、凌乱发丝适合战损、雨夜、情绪崩溃、暧昧镜头。
- 发型作为角色锚点时，跨镜头不要频繁改变长度、刘海和发色。

## 组合模板

```text
[角色身份]，[基础长度]，[发型形态]，
[刘海/分线]，[发色]，[配饰]，
发丝 [质感/动态]，
整体气质 [清爽/干练/御姐/病娇/复古/叛逆]，
与 [服装/妆容/场景] 保持一致。
```

## 避坑

- 不要只写“黑长直”，要补身份、发丝质感、刘海、光线和气质。
- 发型、妆容、服装要属于同一角色逻辑。
- 做角色连续镜头时，发型是重要锚点，不要每镜换一个发型。
- 强风、奔跑、湿发等动态会增加一致性风险，适合单镜头高光，不适合所有镜头。
