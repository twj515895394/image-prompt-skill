# 生图风格选择器

## 定位

本页把“生图风格大全”从长名单重组为可组合的选择器，服务 [[人物外貌与写真Prompt结构]]、[[女友感场景写真Prompt框架]]、[[角色资产生成结构]]、[[场景资产与空间规划]] 和 [[图像风格提取流程]]。如果目标是视频镜头的观看位置或设备感，优先看 [[AI视频风格选择器]]；如果目标是题材级影视气质，优先看 [[影视风格选择器]]。

## 选择顺序

1. 先定输出类型：写真成片、角色资产、场景资产、海报封面、插画叙事。
2. 再定主风格层：东方地域、艺术史、数字设计、幻想科幻、摄影写真、插画绘本、材质工艺。
3. 再定媒介层：超写实、数字油画、水彩、胶片、3D 渲染、矢量平面、版画、刺绣等。
4. 最后只加一个效果层：霓虹、景深虚化、散景、颗粒、体积光、镜像反射等。

## 主风格层

| 主风格层 | 适用场景 | 典型方向 | 使用要点 |
|---|---|---|---|
| 东方地域美学 | 国风写真、武侠角色、东方场景资产 | 水墨写意、新中式、青绿山水、民国旗袍、少数民族 | 重点写留白、矿物色、服饰形制和地域道具，不要只写“国风” |
| 艺术史与绘画流派 | 海报、封面、概念图、题材视觉实验 | 文艺复兴、巴洛克、洛可可、印象派、波普、装饰艺术、极简主义 | 先定构图和色彩方法，再写画派名，避免只借名头 |
| 数字设计质感 | 商业封面、产品图、UI 视觉、潮流海报 | 超写实、概念艺术、3D 渲染、扁平插画、渐变插画、蒸汽波、酸性设计 | 适合明确材质、光效和版式，少和重世界观风格混堆 |
| 幻想与科幻世界观 | 漫剧设定图、场景资产、奇幻封面 | 赛博朋克、蒸汽朋克、黑暗幻想、太空歌剧、废土朋克、克苏鲁 | 先写世界规则，再写装饰元素，避免只堆霓虹和机械词 |
| 摄影写真体系 | 女友感写真、人物海报、生活方式成片 | 棚拍人像、环境人像、街头摄影、复古胶片、中画幅、黑白肖像 | 重点是镜头、肤色、景深、真实细节，适合和 [[电影光影与色调选择器]] 联动 |
| 插画与绘本体系 | 角色立绘、绘本页、二次元封面 | 儿童绘本、日系动漫、美式漫画、商业插画、哥特萝莉、Q 版角色 | 先定线条与上色规则，再定人物比例和表情夸张度 |
| 材质与工艺体系 | 包装、纹样、道具特写、风格化静物 | 木刻版画、拼贴艺术、刺绣、珐琅彩、陶瓷、彩色玻璃、霓虹灯牌 | 更适合作为材料层或局部强化层，不要整页叠多个工艺质感 |

## 组合规则

- 一次只选 1 个主风格层 + 1 个媒介层 + 1 个效果层，避免主风格互相打架。
- 角色资产优先“世界观清楚”，成片海报优先“视觉识别强”，不要用同一套词写所有输出。
- 摄影写真类优先和 [[视角机位与景别选择器]]、[[电影光影与色调选择器]] 联动；插画和概念图优先把笔触、材质、色彩秩序写清楚。
- 风格名必须落到可执行变量：构图、笔触、材质、光线、色调、镜头或空间关系。

## 题材适配

| 题材 | 推荐主风格 | 推荐组合 |
|---|---|---|
| [[女友感写真]] | 摄影写真体系 | 日系清透 / 复古胶片 / 柔焦人像 + 自然光或胶片闪光 |
| [[AI漫剧]] 角色设定 | 东方地域美学、幻想与科幻世界观 | 武侠水墨 / 新中式 / 赛博朋克 + 角色发型与妆容选择器 |
| 场景资产 | 东方地域美学、幻想与科幻世界观、材质与工艺体系 | 青绿山水 / 废土朋克 / 彩色玻璃 + 空间规划与站位 |
| 商业封面或海报 | 数字设计质感、艺术史与绘画流派 | 装饰艺术 / 超写实 / 概念艺术 + 商业色调与版式 |
| 插画叙事 | 插画与绘本体系 | 儿童绘本 / 商业插画 / 哥特萝莉 + 明确线稿与上色方式 |

## 常见失败模式

- 把两个主风格直接并列，例如“水墨 + 波普 + 蒸汽朋克 + 赛博朋克”，最后只剩关键词噪音。
- 把世界观风格和材质效果都写成主体，导致模型不知道重点是题材还是表面特效。
- 只写“高级感、电影感、插画感”，不写留白、颗粒、笔触、材质、镜头或光线。
- 用成片风格去写角色资产，导致角色设定页只有氛围、没有可复用的结构信息。

## 写法模板

```text
输出类型：[写真成片 / 角色资产 / 场景资产 / 海报封面 / 插画叙事]
主风格层：[东方地域 / 艺术史 / 数字设计 / 幻想科幻 / 摄影写真 / 插画绘本 / 材质工艺]
媒介层：[超写实 / 数字油画 / 水彩 / 胶片 / 3D 渲染 / 版画 / 矢量]
效果层：[霓虹 / 景深 / 颗粒 / 体积光 / 镜像反射]
执行变量：[构图、材质、色调、光线、镜头、空间关系]
限制：[不要并列多个主风格，不要只写抽象气质词]
```
