# 分镜资产与 image2 故事版

## 定位

用于生成分镜图、故事版、角色动作拆解图和图生视频参考帧。它连接 [[故事版分镜到视频流程]]、[[图生视频Prompt结构]] 和 [[视角机位与景别选择器]]。

## 7 类 image2 故事版

| 类型 | 用途 | 关键字段 |
|---|---|---|
| 逐镜故事版 | 逐镜生成视频前的画面规划 | 镜号、景别、主体、动作、情绪 |
| 角色故事版 | 保持角色一致性 | 角色锚点、表情、动作、服装 |
| 场景固定故事版 | 固定空间和美术 | 场景、道具、光影、镜头入口 |
| 场景规划故事版 | 设计空间与站位 | 前中后景、站位、动线 |
| 动作拆解故事版 | 拆动作节奏 | 起势、过程、落点、反应 |
| 动作设计故事版 | 设计视觉冲击 | 夸张姿态、速度线、受力反馈 |
| 场景概览故事版 | 交代世界和调度 | 大场景、空间关系、人物位置 |

## 分镜资产字段

```text
镜号：
画面目标：
主体：
场景：
景别/机位：
角色动作：
情绪：
光影色调：
可运动元素：
承接上一镜：
引出下一镜：
```

## 使用原则

- 先做故事版，再写图生视频 Prompt，减少抽卡。
- 分镜图不是最终成片，它的核心价值是“锁定关系、动作和镜头”。
- 图生视频前要标记首帧、尾帧或参考帧的不同作用。
