# 视角机位与景别选择器

## 定位

用于生图、生视频、AI 漫剧分镜和电影感 Prompt。它连接 [[电影感Prompt拆解公式]]、[[分镜资产与image2故事版]]、[[故事版分镜到视频流程]] 和 [[导演级运镜术语选择器]]。

## 景别链

| 景别 | 作用 | 常见用途 |
|---|---|---|
| 远景 | 交代世界和位置 | 开场、旅行、灾难、城市、自然风景 |
| 全景 | 交代人物全身和空间关系 | 站位、动作、群像 |
| 中景 | 建立人物关系 | 对话、互动、冲突前奏 |
| 近景 | 推进情绪和细节 | 表情、手部、道具 |
| 特写 | 锁定证据或情绪爆点 | 眼神、嘴角、伤口、关键道具 |

## 视角与机位

| 类型 | 效果 | 适合场景 |
|---|---|---|
| 平视 | 自然、平等、真实 | 日常写真、对话 |
| 俯拍 | 脆弱、渺小、空间感 | 角色受挫、环境压迫 |
| 仰拍 | 强势、压迫、英雄感 | 反派登场、主角觉醒 |
| 鸟瞰 | 总览、布局、孤立 | 城市、战场、群像 |
| 荷兰角 | 不稳定、失衡 | 悬疑、危机、心理崩塌 |
| POV | 临场、主观 | 追逐、惊吓、第一人称体验 |
| 过肩 | 关系、对话、窥视感 | 双人戏、谈判 |
| 前景遮挡 | 真实、偷拍、层次 | 街拍、卧底、窥视 |

## 镜头参数选择

| 参数 | 方向 | 用法 |
|---|---|---|
| 焦段 24mm | 广角、空间夸张 | 环境、城市、近距离压迫 |
| 焦段 35mm | 纪实、人眼感 | 街拍、日常、短剧 |
| 焦段 50mm | 自然、稳定 | 人像、对话、通用 |
| 焦段 85mm | 压缩背景、肖像感 | 写真、特写、情绪 |
| T1.8 | 浅景深、梦幻 | 人像、浪漫、孤立主体 |
| T2.8 | 兼顾主体和环境 | 常规短剧镜头 |
| T5.6 | 环境清晰 | 群像、空间、动作 |
| 帧率 | 24fps 电影感、60fps 清晰运动 | 按题材选择，不默认堆高 |

## 镜头类型与摄影气质

| 类型 | 效果 | 适合场景 |
|---|---|---|
| 变形宽银幕 | 宽画幅、横向光晕、电影剧照感 | 雨夜、霓虹、赛博都市、电影感封面 |
| 鱼眼镜头 | 边缘扭曲、潮流冲击 | 街头、滑板、演唱会、夸张空间 |
| 微距镜头 | 局部纹理、材质放大 | 食物、花瓣、机械、皮肤、产品细节 |

## 使用原则

- 单镜头通常写 1 个焦段 + 1 个光圈 + 1 个镜头类型就够了。
- 焦段决定空间关系，光圈决定清晰范围，镜头类型决定气质。
- 品牌词如徕卡、哈苏、富士只作视觉提示，不等于真实硬件模拟。
- 避免互斥写法，例如 24mm 广角同时要求强烈长焦压缩，或深景深同时要求背景完全虚化。
