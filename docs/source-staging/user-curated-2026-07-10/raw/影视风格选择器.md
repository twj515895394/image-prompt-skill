# 影视风格选择器

## 定位

本页把“48 种 AI 影视风格”整理成题材级组合模板，用来先决定一部作品像什么，再回到 [[AI视频风格选择器]] 选择设备、观看位置与具体镜头。它服务 [[写实电影短剧]]、[[AI漫剧]]、[[电影感Prompt拆解公式]] 和 [[故事版分镜到视频流程]]。

## 与 AI 视频风格选择器的分工

- 本页回答：题材叙事、作者气质、时代质感、世界观规模。
- [[AI视频风格选择器]] 回答：观看位置、设备媒介、具体镜头质感、落 Prompt 的方式。
- 先用本页定“像哪类片”，再用视频风格页定“怎么拍出来”。

## 选择顺序

1. 先定现实度：现实观察、作者化现实、史诗仪式、废墟哲思、奇幻世界。
2. 再定情绪主轴：孤独、压迫、克制、秩序、诗意、奇观。
3. 再定镜头锚点：玻璃反射、低角度仰拍、高角度俯拍、荷兰角、对称构图、自然光、长镜头等。
4. 最后才补设备与镜头执行：固定机位、手持、长焦、慢镜头、CCTV、红外、FPV 等。

## 题材级风格模板

| 风格模板 | 核心组合 | 适用题材 | 使用要点 |
|---|---|---|---|
| 都市孤独现实 | 玻璃反射 + 夜景霓虹 + 长焦压缩 + 克制人物关系 | 都市情绪、暧昧关系、夜行短片 | 用反射、雨夜和留白建立关系距离，不要把霓虹写成赛博世界观 |
| 家庭克制现实 | 自然光 + 日常空间 + 稳定镜头 + 收敛表演 | 家庭关系、成长、生活流短剧 | 让空间和表演说话，减少炫技镜头，适合接 [[电影光影与色调选择器]] 的暖黄或中性色调 |
| 权力史诗与仪式 | 低角度仰拍 + 对称构图 + 大色块秩序 + 群像调度 | 历史题材、权谋、典礼、登场 | 重点是秩序和体量，不要只剩“红绿撞色”这种单一视觉噱头 |
| 废墟哲思与末世诗意 | 长镜头 + 水面反射 + 废墟空间 + 低饱和冷调 | 末世、意识流、回忆、哲思影像 | 空间要有时间感和潮湿感，镜头节奏必须慢下来 |
| 群像戏剧与对峙 | 高反差黑白 / 冷灰现实 + 固定机位 + 群像 blocking | 武士对峙、帮派冲突、戏剧性群像 | 先设计站位与视线关系，再决定切镜频率 |
| 奇幻史诗与世界观展示 | 写实质感 + 巨构空间 + 山水 / 城堡 / 遗迹 + 奇观尺度 | 漫剧封面、奇幻短片、世界观海报 | 适合先写空间和文明痕迹，再写人物，不要直接堆满魔法名词 |
| 秩序怪诞与作者化空间 | 对称空间 + 设计感布景 + 粉彩或怪诞暗色 + 冷静镜头 | 怪诞寓言、黑色幽默、舞台化广告 | 画面重点在人工布置感，演员、道具和色块要一起服从秩序 |

## 组合规则

- 一条成片链路通常使用 1 个题材模板 + 1 个作者气质 + 1 个设备层风格，不建议三层都选多个。
- [[AI漫剧]] 更适合“权力史诗 / 废墟哲思 / 奇幻史诗 / 群像戏剧”四类模板；[[写实电影短剧]] 更适合“都市孤独现实 / 家庭克制现实 / 群像戏剧”。
- 如果已经引用强作者气质，就少加新世界观词；如果先用世界观模板，就让作者气质只做微调。
- 静态海报可以只写题材模板和色调；视频镜头必须继续补观看位置、运动方式和剪辑节奏。

## 常见失败模式

- 只写导演名，不写对称、长镜头、反射、低角度等可执行视觉变量。
- 同时叠“作者气质 + 设备媒介 + 世界观流派 + 色调滤镜”四层主锚点，导致每层都弱。
- 把写实家庭戏写成高强度奇观风格，或把史诗场面写成过度日常化的平光纪录感。
- 忘了题材页和执行页分工，直接把本页当成最终 Prompt 模板。

