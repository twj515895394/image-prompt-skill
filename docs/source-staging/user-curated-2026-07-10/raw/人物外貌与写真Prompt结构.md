# 人物外貌与写真 Prompt 结构

## 定位

用于写实写真、AI 漫剧角色成片、女主形象和角色定妆图。它连接 [[基础Prompt结构]]、[[角色资产生成结构]]、[[角色发型选择器]]、[[女主妆容选择器]]、[[表情与微表情选择器]] 和 [[女友感场景写真Prompt框架]]。

## 结构

```text
年龄身份 + 整体气质 + 脸型五官 + 皮肤质感 + 发型 + 体态 + 服装材质 + 场景 + 光影镜头
```

## 变量表

| 维度 | 写法重点 | 避坑 |
|---|---|---|
| 年龄身份 | 年龄段、职业、角色关系 | 不要写模糊身份堆叠 |
| 整体气质 | 清冷、温柔、锋利、松弛、元气 | 气质要和妆容、服装一致 |
| 脸型五官 | 脸型、眉眼、鼻梁、唇形、下颌线 | 避免同时给互斥脸型 |
| 皮肤质感 | 自然皮肤、毛孔、微瑕、柔光 | 写实写真不要过度磨皮 |
| 发型 | 长短、卷直、刘海、发色、湿发/凌乱感 | 接入 [[角色发型选择器]] |
| 妆容 | 底妆、眉眼、唇色、腮红、角色气质 | 接入 [[女主妆容选择器]] |
| 表情 | 眼神、眉眼、嘴角、呼吸、小动作 | 接入 [[表情与微表情选择器]] |
| 体态 | 肩颈、站姿、坐姿、手部状态 | 写清手部，减少畸形 |
| 服装材质 | 衬衫、针织、丝绸、牛仔、薄纱 | 材质和场景温度要一致 |
| 光影镜头 | 景别、焦段、光线、景深 | 接入 [[视角机位与景别选择器]] |

## 推荐写法

```text
[年龄身份]，[整体气质]，
[脸型五官]，[皮肤质感]，
[发型]，[体态与动作]，
穿着 [服装与材质]，
位于 [场景]，
[景别/机位/焦段]，[光影]，[色调]。
```

## 常见失败模式

- 只写“美女”“高级脸”，没有脸型、五官和气质约束。
- 发型、妆容、服装、场景不属于同一审美方向。
- 写实写真没有皮肤细节，导致塑料感。
- 没有手部和体态约束，容易出现肢体问题。
