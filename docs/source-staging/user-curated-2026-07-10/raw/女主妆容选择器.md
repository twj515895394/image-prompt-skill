# 女主妆容选择器

## 定位

用于女主成片、角色定妆图、AI 漫剧封面和人物写真。妆容是角色气质开关，应和 [[角色发型选择器]]、[[人物外貌与写真Prompt结构]]、[[电影光影与色调选择器]] 同步选择。

## 妆容 Prompt 结构

```text
人设气质 + 底妆质感 + 眉眼 + 眼线/眼影 + 腮红/唇色 + 光影色调 + 服装场景
```

## 避坑

- 妆容不能只写“好看”，要说明底妆、眼妆、唇色和角色气质。
- 甜妹妆和暗黑女王妆不要混用，除非明确要反差角色。
- 职业角色优先写干净、克制、利落，不要过度舞台妆。
- 病娇和战损妆容易生成病态过度，需要控制“苍白、泪痕、红眼尾”的强度。
