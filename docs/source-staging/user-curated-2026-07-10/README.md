本目录暂存用户在 2026-07-10 提供的一批生图、角色、场景、分镜、镜头、光影与风格资料，用于后续查重、拆分、提炼和迁移。

## 目录

```text
docs/source-staging/user-curated-2026-07-10/
```

## 处理原则

1. 先查重，再决定更新现有叶子或新建叶子。
2. 默认优先更新现有正式 Reference，不建立第二份正文真源。
3. 混合职责资料必须拆分后再分别进入 `tasks/`、`controls/`、`libraries/` 或 `styles/`。
4. 大型综合选择器只作为迁移来源，不直接成为运行时万能 Reference。
5. 正式迁移后补索引和路由验证。
6. 本批处理完成后，`raw/` 可删除；长期保留来源审计和最终映射记录。
