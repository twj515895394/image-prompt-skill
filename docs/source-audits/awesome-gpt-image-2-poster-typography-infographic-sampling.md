## 2. 抽样类型

抽样覆盖以下代表方向：

- 单主视觉电影海报；
- 产品发布和品牌活动海报；
- 人物主导的社交媒体宣传图；
- YouTube 缩略图和高点击封面；
- 标题字主导的字体海报；
- 图文混排品牌提案板；
- 教育型信息图；
- 时间线、步骤图和对比图；
- 数据卡片和指标面板；
- 地图、流程和结构示意图；
- 食品、产品和活动 Banner；
- 多模块社交平台长图。

每个正式结论均来自多种不同题材和版式的重复规律，而不是单个案例创意。

## 3. 跨案例稳定规律

### 3.1 信息层级先于装饰风格

高质量图文设计通常先确定：

```text
主标题
→ 主视觉
→ 副标题或核心卖点
→ 辅助信息
→ 品牌 / 行动信息
```

字体、颜色、光效和装饰只能强化层级，不能代替层级。

### 3.2 海报版式存在可复用结构

重复出现的结构包括：

- 主体居中 + 上下文字；
- 主体偏侧 + 对侧文字区；
- 大标题压图；
- 上图下文；
- 左图右文；
- 对角动势；
- 分栏网格；
- 卡片矩阵；
- 留白极简；
- 满版背景 + 局部信息块；
- 缩略图式人物 / 产品大特写 + 少量大字。

结构应由内容密度、平台比例和阅读距离决定。

### 3.3 文字可读性依赖空间和对比

稳定规律包括：

- 标题区背景要相对平整；
- 文字与背景需要明度、色相或描边分离；
- 行数、字数和字号要受控；
- 小字不应放在高频纹理或强反光区域；
- 中文、英文和数字需要明确字重层级；
- 图像生成任务中，文字准确性应被单独声明和校验。

### 3.4 信息图需要“结构语义”

有效信息图不是把图标平均排开，而是先确定：

- 比较；
- 流程；
- 时间线；
- 层级；
- 分类；
- 地理关系；
- 因果关系；
- 数据指标；
- 组成结构。

视觉结构应与信息关系一致。

### 3.5 缩略图是高压缩海报

缩略图的稳定规律：

- 一个主主体；
- 一个强情绪或强产品卖点；
- 3–8 个主要文字字符或极短标题；
- 高明度或色彩对比；
- 移动端缩小后仍可辨识；
- 背景细节受控；
- 不依赖小字说明。

因此缩略图归入海报版式选择，而不是独立 style。
